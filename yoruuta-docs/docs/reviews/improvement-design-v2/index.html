よるのうたかた 大規模改善設計書 v2

目標​

ペルソナ評価で「ストーリー構成: 7」「感動・情緒: 8」だったスコアを 9以上 に引き上げる。 Key作品（CLANNAD, AIR, Kanon）と同等もしくはそれ以上の伏線・感動を目指す。

1. 選択肢構造の再設計​

現状分析​

  • 全15箇所の選択肢: 2択1箇所、3択13箇所、4択1箇所
  • 3択が87%を占め、単調
  • 正解が明白（「丁寧・寄り添い型」が常にプラス）
  • 沈黙選択肢が皆無
  • 序盤（ch00全体+ch01前半）に選択肢なし
  • 致命的問題: ch04/scene_04のルート分岐が「誰を追うか」の明示的選択になっている
    • これでは好感度システムが単なるゲートチェックになり、蓄積の意味が薄れる
    • プレイヤーが「自然とそばにいた」と感じられない

改善案​

根本方針: 蓄積型ルート決定への移行​

ルートはプレイヤーの明示的な人物選択ではなく、蓄積された好感度・信頼・関わりの結果として自動決定する。

施策対象効果
ルート分岐を態度選択に変更ch04/scene_04「誰を追うか」→「状況にどう向き合うか」に変更。ルートは好感度の自動判定で決まる
3択→2択に変更ch04/scene_03, ch11/scene_04, ch12/scene_04二者択一で価値観を問う
新規2択追加ch01/scene_03 L74付近, ch02/scene_05 L158付近感情投資の早期化
沈黙選択肢導入ch03/scene_01, ch03/scene_02「何も言わない」が正解になる場面
テキスト台詞化全選択肢態度説明→朔の実台詞に統一
善意トラップ設計ch12/scene_04「助けたい」がBADになる構造
サブストーリー好感度効果湊・いろは・水城先生サブサブキャラとの交流が間接的にヒロイン好感度に影響

ch04/scene_04 新選択肢設計​

旧: 「校舎の裏手へ」「正門から、まっすぐ」「東門の方へ」（人物の直接選択）

新: 「企画書を取り戻しに行く」「少し、頭を冷やそう」「今は、みんなの方が大事だ」（態度の選択）

  • 態度選択は好感度を ±1 調整し、project_intact に影響する
  • ルート自体は蓄積された好感度の比較で自動決定される
  • @if チェーンによる資格判定 → 好感度比較 → タイブレーク処理

ルート自動決定アルゴリズム​

  1. project_intact チェック → false なら共通BAD
  2. 各ヒロインの資格判定: (trust_step1 AND affinity >= 3 AND saw_core) OR (affinity >= 6 AND saw_core)
  3. 資格ありヒロインの中で最も好感度が高い方のルートへ
  4. 同率時: trust_step1 所持 → saw_core 取得順序 でタイブレーク
  5. 全員高水準で資格あり → True Route フラグをセット

サブストーリーによる好感度効果​

  • 湊 sub_01（ch01後）: 結月の話題が出る → yuzuki +1
  • いろは sub_04（ch02後）: 咲夜の内面を観察 → saya +1
  • いろは sub_05（ch04後）: 千紘の想いの痕跡 → chihiro +1
  • 水城先生 sub_06（ch03後）: saw_core 理解の補助

改善後バランス: 2択35% / 3択65%（ルート分岐の4択を3択態度選択に変更）

2. 感動・伏線設計の強化​

最優先施策​

2-1. 朔の音声メッセージの段階的展開（6箇所追加）​

挿入位置内容回収先
ch00 scene_01「再生時間2:47/8:12」通知を見て閉じるTrue Route scene_05
ch01 scene_04帰路で古いスマホを鞄の底で触る描写千紘ルート
咲夜ルート scene_03朔が過去を告白（共鳴構造）咲夜「聞いてて」との対比
千紘ルート scene_02千紘の電話に出る=「今度は聞く」朔のテーマ回収
結月ルート scene_04音声の全容判明: 最後の言葉は「ありがとう」最も痛い回収
True Route scene_058:12の全体を聞く。「話したいことがある」で終わる作品テーマ完結

2-2. 各ルートの「決壊」ポイント​

ルート現状追加する「決壊」
咲夜静かな余韻のみ翌朝、凪への原初の歌を歌い終え涙。「止め方が分からない」
千紘精算書で着地紙コップの箱を捨てようとして捨てられない。「なんで重いのよ」
結月四行朗読で着地奨学金書類を開き、穏やかな笑顔を作ろうとして失敗。ペンを借りて返さない

2-3. 日常シーンの二重意味化​

序盤の描写表面の意味終盤で判明する裏の意味
結月「うち、しょっぱいんだよね」味の好み母の入院後、自炊で味付けが変わった
結月「払えるうちに払いたい」几帳面な性格家計問題の反映
千紘が校門でスマホを確認して戻す連絡確認「帰りたくない」
朔が鍵を返すのを忘れるうっかり無意識に学校に居場所を作ろうとしている

構造強化​

  • ルート間連鎖: 咲夜の歌の「帰りたくない」= 結月の朗読の「まだ帰りたくないから、歩いている」
  • ヒロイン間対話: 咲夜×結月「止められないのも止まってるのも、自分では選べない」
  • 共通BADに御子柴視点追加: 一年生が先輩たちの失敗を片づける構図
  • 湊視点シーン: 渡り廊下から小講堂を見て「お前、変わんないまま変わってんだよな」

5. キャラクター関係性マップ​

朔と各キャラの「鏡」構造​

キャラ共鳴する傷
咲夜朔は「聞かなかった」、咲夜は「届けてしまった」。方向は逆だが「声と相手の間の責任」が同じ
千紘朔は音声メッセージを一人で抱える。千紘は家庭の負荷を一人で抱える。「頼ることへの恐怖」
結月朔は過去の後悔に未来を閉じかけている。結月は家庭事情に未来を閉じている。「動けない」
湊朔は重く待つ。湊は軽く先に動く。方法が違うが「居場所を失う恐怖」が根
いろは朔は聞く側、いろはは支える側。自分を前に出さないことで安全を保つ
真帆朔が聞けなかったことを真帆は自然にやっている。朔の理想形

ヒロイン間の関係性（追加すべき交流）​

組み合わせ対比テーマ追加シーン案
咲夜×千紘沈黙 vs 攻撃ch03: 「あなたの歌詞、甘い」「尖らせるのが正しいとも思わない」
結月×湊停止 vs 運動ch02: 「疲れないの？」「疲れるよ。でも止まると考えるだろ」
咲夜×結月表現者 vs 受け手ch02: 咲夜の文章に結月が「これ、好き」
千紘×結月責任 vs 放棄ch03: 「興味を持ったものが消える方が怖い」→千紘が黙る
