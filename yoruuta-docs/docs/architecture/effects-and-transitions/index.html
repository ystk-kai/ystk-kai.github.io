エフェクト・トランジション定義

シナリオ中で使用可能なエフェクトおよび背景トランジションの一覧。

エフェクト（@effect）​

シナリオコマンド @effect <name> [duration] で発火する画面エフェクト。 CSS アニメーションで実装されており、指定時間経過後に自動的に解除される。

エフェクト名デフォルト持続時間説明
shake400ms画面全体を揺らす。衝撃・地震・動揺の演出
flash300ms白い光がフラッシュする。閃光・雷・記憶のフラッシュバック
fadeout_all1500ms画面全体が黒くフェードアウトする。暗転
fadein1500ms黒画面から明るくなる。暗転解除
whiteout800ms画面が白くなる。ホワイトアウト
whitein800ms白画面から元に戻る。ホワイトアウト解除
crossfade1200ms画面が一度暗くなり再び明るくなる。シーン転換
checker_in1200ms市松模様タイルで画面を閉じる。エンド遷移向け
checker_out1200ms市松模様タイルで画面を開く。開始遷移向け

持続時間のカスタマイズ​

持続時間は位置引数または duration= で指定可能。省略時はデフォルト値を使用する。

@effect fadeout_all              # デフォルト 1500ms
@effect fadeout_all 2000         # 位置引数で 2000ms に指定
@effect fadein duration=1000     # 名前付き引数で 1000ms に指定

シナリオでの使用例​

@effect shake
@effect flash
@effect fadeout_all duration=2000
@effect fadein duration=1000
@effect checker_in duration=3000
@effect checker_out duration=1800

実装ファイル​

  • エフェクトコンポーネント: apps/web/src/components/game/EffectLayer.tsx
  • CSS アニメーション: apps/web/src/components/game/EffectLayer.module.css
  • shake エフェクト（ステージ全体）: apps/web/src/components/game/GameScreen.module.css
  • デフォルト持続時間定義: apps/web/src/store/gameStore.ts (EFFECT_DURATION)

背景トランジション（@bg）​

背景画像の切り替え時に適用されるトランジション効果。 位置引数と名前付き引数の両方をサポートする。

トランジション名デフォルト持続時間説明
instant0ms即座に切り替え（デフォルト）
fade800msフェードで切り替え
dissolve1500msディゾルブで切り替え
slide-left600ms左方向にスライドして切り替え
slide-right600ms右方向にスライドして切り替え

構文​

@bg <image> [transition] [duration]
@bg <image> transition=<type> [duration=<ms>]

シナリオでの使用例​

# 位置引数スタイル
@bg classroom_evening fade
@bg hallway dissolve 2000

# 名前付き引数スタイル（推奨）
@bg classroom_evening transition=fade
@bg hallway transition=dissolve duration=2000
@bg school_gate transition=slide-left duration=800

実装ファイル​

  • 背景コンポーネント: apps/web/src/components/game/BackgroundLayer.tsx
  • CSS トランジション: apps/web/src/components/game/BackgroundLayer.module.css
  • デフォルト持続時間定義: apps/web/src/store/gameStore.ts (TRANSITION_DURATION)

シーン転換マクロ（@scene_change）​

暗転→背景変更→明転を1コマンドで実行するマクロコマンド。 コンパイル時に3つのコマンド（effect + bg + effect）に展開される。

構文​

@scene_change <image> [effect=<black|white>] [duration=<ms>]
パラメータ説明デフォルト
image切り替え先の背景画像（必須）-
effect暗転の種類: black（暗転）or white（ホワイトアウト）black
duration暗転・明転それぞれの持続時間 (ms)エフェクトのデフォルト値

シナリオでの使用例​

# 暗転 → 背景変更 → 明転（デフォルト）
@scene_change night_sky

# ホワイトアウトでシーン転換
@scene_change classroom effect=white

# 持続時間を指定
@scene_change hallway duration=800

# ホワイトアウト + 持続時間指定
@scene_change rooftop effect=white duration=600

展開例​

@scene_change night_sky duration=800 は以下の3コマンドに展開される:

{ type: "effect", name: "fadeout_all", duration: 800 },
{ type: "bg", image: "night_sky" },
{ type: "effect", name: "fadein", duration: 800 },

画面遷移エフェクト​

ゲームのフェーズ切り替え時に自動適用されるエフェクト。

遷移エフェクト説明
タイトル → ゲーム開始フェードアウト → フェードインタイトル画面が 0.8s で暗転し、ゲーム画面が 1.5s でフェードインする

実装ファイル​

  • タイトルフェードアウト: apps/web/src/components/ui/TitleScreen.module.css (.fadeOut)
  • ゲーム画面フェードイン: apps/web/src/store/gameStore.ts (loadChapter)

キャラクター表示アニメーション​

キャラクターが画面に表示される際の演出。

アニメーション持続時間説明
charaEnter500msフェードイン + 軽い上方向スライドで登場
charaBreathe4s (無限ループ)微小な scale 変化 (1.0 → 1.004) で呼吸を表現。登場アニメーション完了 0.5s 後に開始
  • charaBreathe は prefers-reduced-motion: reduce が設定された環境では無効化される
  • scale ベースの微動は translateY と比較して「画像のガタつき」感が少なく、静止PNGに対して自然な動きを実現する

実装ファイル​

  • apps/web/src/components/game/CharacterLayer.module.css (.visible)

スピーカーカラー​

テキストウィンドウの話者名は、キャラクターごとのテーマカラーで着色される。 CharacterDefinition.themeColor に設定された色が CSS 変数 --speaker-color として適用される。

キャラクターテーマカラー視覚的印象
咲夜#d4a8e0淡紫
千紘#8ec8e8淡青
結月#e8c090暖橙
あかり#e8a0b8淡桃
湊#90d0b8淡翠
いろは#d8c890淡金
水城先生#a8b8d0灰青
その他 / 未設定#e8d8b8フォールバック（従来色）
  • カラーマップは defaultName、nameOverrides の各 displayName、id をキーとして逆引きで構築される
  • themeColor 未設定のキャラクターやNPCには CSS のフォールバック色 #e8d8b8 が適用される
  • テーマカラーは暗い背景 (--color-window-bg) に対して WCAG AA (4.5:1) のコントラスト比を満たす色を選定すること

実装ファイル​

  • テーマカラー定義: apps/web/src/data/characterDefinitions.ts (themeColor)
  • カラー適用: apps/web/src/components/game/TextWindow.tsx (speakerColorMap)
  • CSS 変数: apps/web/src/components/game/TextWindow.module.css (.speaker)

プロパティ通常ナレーション (>)内心テキスト (>>)
不透明度1.00.72
  • ストア (gameStore) の isInnerThought フラグで管理される
  • セーブ/ロード、シナリオ終了時に false にリセットされる
  • 構文仕様の詳細は シナリオ表記ガイド を参照

  • パーサー: packages/scenario-compiler/src/parser.ts (inner_thought 行タイプ)

  • スタイル: apps/web/src/components/game/TextWindow.module.css (.innerThought)

インラインポーズ​

テキスト中の {p:N} 構文により、文字送りアニメーション中に N ミリ秒の待機を挿入する。

  • ルビ構文 {親文字|ルビ} とは | の有無で区別され、競合しない
  • スキップモード時はポーズを無視して即時表示する
  • 同一位置に複数のポーズが連続した場合は待機時間を加算する
  • 構文仕様の詳細は シナリオ表記ガイド を参照

実装ファイル​

  • テキストパーサー: apps/web/src/utils/rubyParser.ts (parseRubyText, computePauseMap)
  • アニメーション: apps/web/src/hooks/useTextAnimation.ts (pauseUntilRef)

パフォーマンスに関する注意​

  • すべてのエフェクトは CSS アニメーションで実装し、JavaScript による DOM 操作を最小限にしている
  • will-change: opacity / will-change: transform を適切に設定して GPU アクセラレーションを活用
  • エフェクト再生中（isEffectPlaying: true）はテキスト送り操作を無効化し、不整合を防止
  • charaBreathe は無限ループだが、CSS アニメーションのため GPU compositing で処理され、メインスレッドへの負荷は最小限
  • prefers-reduced-motion: reduce 環境ではループアニメーション（呼吸）を無効化する
