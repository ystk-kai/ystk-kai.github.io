ID の意味づけと場面意図については 演出対応表 を参照すること。

ファイル命名規則​

アセットファイルは以下の規則に従って配置する。

assets/
├── bg/          # 背景画像
│   ├── school_corridor_evening.webp
│   ├── classroom_morning.webp
│   └── ...
├── bgm/         # BGM
│   ├── empty_school.ogg
│   └── ...
├── se/          # 効果音
│   ├── floor_creak.ogg
│   └── ...
└── chara/       # キャラクタースプライト
    ├── saya/
    │   └── neutral.webp
    ├── chihiro/
    │   └── neutral.webp
    └── ...
  • 背景・BGM・SE のファイル名はシナリオ内の ID とそのまま一致させる
  • キャラクタースプライトはキャラクターID のディレクトリ内に表情名で格納する
  • 画像形式は WebP、音声形式は OGG を基本とする
