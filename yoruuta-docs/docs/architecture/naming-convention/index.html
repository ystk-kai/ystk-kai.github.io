チャプターID・END名 命名規則

目的​

シナリオファイルのチャプターID、ENDラベル、到達フラグの命名規則を定義し、 帯域ごとの役割を明確にする。

BAD END シーンラベル​

BAD END の個別シーンは以下のパターンで命名する。

scene_bad_{識別名}
ラベルルート内容
scene_bad_silence咲夜完全な沈黙
scene_bad_severed千紘関係断絶
scene_bad_vanish結月静かな消失

これらは変更しない（既に統一されている）。
