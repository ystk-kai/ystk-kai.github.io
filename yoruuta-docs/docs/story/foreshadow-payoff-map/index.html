伏線・回収対応表

目的​

本資料は、共通ルートから個別ルート、BAD END までの伏線と回収を一覧化する。 短い台詞や小物が終盤でどう意味を変えるかを管理し、感動の核が説明不足にも説明過多にもならないようにする。

主人公の軸​

伏線初出回収意味
聞ける時に聞かなかった後悔プロローグ共通BAD、各正規ルート聞く姿勢そのものが主人公の選択軸
急いで答えを出さないプロローグ、第3章各ルート終盤の正解選択助けるより先に、相手の言葉を置く

タイトル「よるのうたかた」の伏線構造​

タイトルは作品最大の伏線。詳細設計は 感情深度の改善設計 改善3 を参照。

伏線要素初出回収意味
「夜」と本音の結合プロローグ（旧音楽室の夜の空気）各ルートクライマックス（夜の中で声を出す決断）夜＝隠された感情の時間。その中で声にすることがタイトルの核
「声にしない」の反復第1章〜第7章（咲夜の沈黙、千紘の拒絶、結月の回避）各ルート終盤（それぞれの形で声にする）「うたかた」の中の「うた」＝声にする行為。声にしない蓄積が、声にする瞬間の重みを作る
「消える」の予感第2章（紙コップ初登場）第4章（紙コップの箱が片付けられる＝伏線深化）、第7章（失われる直前の頂点）、各ルートED（形を変えて残る）泡沫＝消えるもの。だが声にしたものは消えない。タイトルの二重性
ひらがなの不確定性タイトル画面全ルートクリア後のタイトル画面帰還漢字を当てられないこと＝言葉にできない感情の構造そのもの

日常モチーフ​

モチーフ初出回収意味
机の下のマーカー第1章第3章冒頭、AルートED普通に言える関係へ戻る変化
紙コップと砂糖の軽口第2章第4章破綻直前（箱が片付けられている）、各ルートEDで変形再登場笑えた空気が壊れることと、形を変えて取り戻せる平熱
「同じだね」から始まる共同作業第1章CルートED受け身の共有から、自分で次を選ぶ共有への変化

Aルート 綾瀬川咲夜​

伏線初出回収意味
誰にも見せたくない言葉第1章本番導入、AルートED隠す言葉から、自分の責任で渡す言葉へ移る
見せる形にすると違うものになる第2章Aルート終盤の選択表現への恐れを消さずに引き受ける
聞かなかったことにしてプロローグAルートED見えていても黙る関係から、見えていることを言葉にする関係へ変わる
咲夜 の読みが宙づりのまま残るプロローグ翌朝、中盤のノート本番後の名呼び正しく呼ばれることを、咲夜自身が受け取れるようになる

Bルート 四辻千紘​

伏線初出回収意味
雰囲気だけの案は通さない第1章Bルート終盤の役割分担管理ではなく、守るための線引きへ変わる
一人で片づけた方が早い第2章、第3章Bルート終盤の台本手渡し抱え込みから委譲への転換
会計どうするの第2章BルートED実務感覚を失わずに他者から受け取る変化

Cルート 穂坂結月​

伏線初出回収意味
終わりが決まってる作業は少し楽第2章CルートED終わりの見える安全圏から、未確定の未来へ一歩出る
進路の話題を避ける第1章、第3章CルートED回避から、自分から書類を見ると言う段階へ変わる
欲しいものを口にしない共通ルート全体Cルート本番の四行大きな夢ではなくても、現在形の意思を言葉にする
結月の四行を殊羽が預かるCルート終盤（紙コップの裏に書いて投書箱へ）C隠し（結月「出してきた」）第4章で壊れた投書箱の信頼が、結月の意思と殊羽の誠実さで静かに修復される

サブヒロインルート​

殊羽ルート​

伏線初出回収意味
「投書に何書いたかは、聞かんよ」第2章（投書箱設置時）殊羽ルート終盤（自分の言葉を初めて出す）預かる側だった殊羽が、声にする側に変わる
殊羽の天然が消える第4章以降殊羽ルート後半（天然が戻るが第7章ほどには戻らない）元気さの喪失と部分的回復。完全復帰しないことが日常の変化の証
殊羽の方言バロメーター第2章〜第7章殊羽ルートで博多弁が自然に戻る瞬間素の言葉に戻れたことが回復の証拠

あかりルート​

伏線初出回収意味
あかり「また『どっちでもいい』出てるよー」第2章〜あかりルート（結月が動き出した後、合図を送る相手がいなくなる）支える側の役割が消えた時の喪失
結月が「あかちゃん」と呼ぶ中学時代の癖あかりルートED（呼び方の意味が変わる）「結月の友達」ではない、あかり個人への視線
あかり「久瀬くん、優しいのは分かるけどさ」第3章あかりルート（あかり自身も同じことをしていたと気づく）支える側だからこそ分かる「踏み込みすぎ」の構造

いろはルート​

伏線初出回収意味
いろは「先輩たちは、もう緊張してます」第2章いろはルート（見守っていた側の声が初めて届く）外部視点の客観性が、いろは自身の声に変わる
中学時代に聞いた声の記憶いろは背景設定いろはルート（咲夜の声と繋がる）届ける側に回った動機の根源
いろは「几帳面じゃなくて、怖いだけです」第2章いろはルートED（怖さを認めた上で、届ける側を選び直す）距離を置くことの本当の理由

第4章の圧の伏線（改善5）​

伏線初出回収意味
咲夜の匿名活動への言及第1章（さりげなく）第4章（特定される）隠すことで守っていたものが、隠せなくなった時の恐怖
千紘の電話に出ない習慣第2章第4章（朔の前で出ざるを得ない）管理できる範囲が崩れる瞬間
結月の書類の棚上げ第1章（進路を避ける）第4章（期限切れを公に指摘される）止まっていることの代償が形になる

サブキャラクター経由の伏線（湊・真帆・水城）​

伏線初出回収意味
真帆「最近楽しそうだね」第5章第7章で2回目の指摘、各ルートBADでは3回目が来ない外から見た幸福の証拠。失われると外からも分かる
湊「待てるけど待ちすぎる」第1章（朔への指摘）各ルート中盤で朔が踏み込み方を誤る瞬間、正規EDで「待ち方」が変わる朔の弱点と成長の軸
水城「本音が正しいとは限らない」第3章各ルートで本音を言わせることが解決にならない場面主人公の安易な「聞けば救える」を崩す
いろは「先輩たちは、もう緊張してます」第2章True Route、友情END年下から見た上級生たちの危うさ。外部視点の客観性

殊羽経由の伏線​

伏線初出回収意味
殊羽「投書に何書いたかは、聞かんよ」第2章（投書箱設置時、素朴な宣言として）第4章（投書が軽く消費される）、Cルート（結月の四行を黙って預かる）預かった言葉への誠実さ。壊された信頼が、結月の四行を通じて静かに修復される
殊羽の天然・ドジによる笑い第2章〜第7章（あかり・湊・いろはとの掛け合い）第4章以降（いじられても反応が鈍い。あかりが振っても空振り）場を最も明るくしていた存在の沈黙が、全体の温度低下を可視化する。失われて初めて価値が分かる
殊羽の方言が薄くなる第2章〜第7章（博多弁＝リラックスの証）第4章（投書消費後、標準語に寄る。緊張・自責のバロメーター）方言の切り替えが殊羽の内面の指標。素の博多弁が戻ることが回復の証拠
あかり「ことはちゃんさあ……」（定型のツッコミ導入）第2章〜第7章（殊羽のドジへの即座ツッコミ）第4章以降（振っても殊羽が空振り。定型が崩れる）、各ルート後半（少しずつ噛み合いが戻る）日常の掛け合いの健全さの指標。このやり取りが戻ることが場の回復を可視化する

掛け合いコメディの伏線​

伏線初出回収意味
朔×湊の軽口の応酬第1章〜第7章（日常の掛け合い）True Route（湊が冗談を挟まず核心を突く）軽口のテンポが途切れることで、事態の深刻さを体感させる
千紘の「予定通りに遅れている」第2章（企画ブレスト）Bルート（管理する余裕すらない状態）ドライユーモアの消失が千紘の限界を示す
結月の全員の砂糖の量第7章（紙コップ休憩）Cルート（「どっちでもいい」すら言わなくなる）止まっていても見ていた。その観察力が消えることの喪失
朔の天然ボケ（本人無自覚）第2章〜第7章（散発的に）第4章以降（朔自身に余裕がなくなり天然が出ない）主人公の人間味の喪失。場の空気が硬くなる指標
全員が笑う紙コップ休憩第7章（幸福の頂点）第4章（小講堂の沈黙）、各ルートED（形を変えて再現）全員の声が揃った最後の瞬間。二度と同じ形では戻らない

共通BAD での反転​

失われるもの元の置き場BAD での見せ方意味
教室の軽口第1章テープ跡だけが残る普通の時間を守れなかった
紙コップの休憩第2章小講堂の箱だけが残る企画以前に人の心が壊れた
旧音楽室の気配プロローグ、第3章扉の向こうが無音になる聞くべき時に聞かなかった帰結

紙コップの象徴設計​

紙コップは作品全体を通して意味を変えながら再登場する中心小道具。

登場章状態意味
初登場第2章砂糖の量、会計の軽口企画チームの日常が始まる
定着第5章結月が全員の砂糖の量を覚えている止まっていても周りを見ていた証拠
幸福の頂点第7章全員の素の掛け合い、最も温かい空気失われる前の最後の平熱
喪失第4章紙コップの箱が片付けられている壊れた日常の象徴
A回収Aルート終盤咲夜が本番前に一人で紙コップにお茶を入れている。朔の分も声を出す決意の前の、静かな日常への回帰
B回収Bルート終盤千紘が「これを任せる」と言う時、手に紙コップの会計メモ管理を手放す瞬間に、管理の道具が委譲の証拠になる
C回収Cルート終盤結月が四行の原稿を紙コップの裏に書いている「きちんとした紙に書く＝正式に提出する」のハードルを超えられない結月が、捨てられる紙コップの裏だからこそ書けた。日常の小道具が、初めての意思表示の媒体になる

設計原則​

  • 紙コップそのものに注目させすぎない。あくまで場面の中の自然な小道具として置く
  • ルートごとの回収で「同じ小道具が違う意味を持つ」体験をさせる
  • プレイヤーが3ルート通した時に初めて「紙コップ」の反復に気づく設計

水城の二つの問い​

問い初出回収意味
「本音を言うことが正しいとは限らない」第3章各ルートで本音を言わせることが解決にならない場面主人公の安易な「聞けば救える」を崩す
「聞くことと、何もしないことは、どう違う？」第3章True Route（朔が「聞く側」を選び直す場面）朔の「聞く側」というアイデンティティへの揺さぶり

湊の沈黙の蓄積​

場面章湊の反応回収先
朔が相談を受けた後第2章「お前、飯食ったか？」→朔が「大丈夫」→追わないTrue Route
日常会話の中で第5章「お前って、自分の話あんまりしないよな」→朔が話題を変える→引くTrue Route
帰り道の別れ際第7章「じゃあな」の後に何か言いかけてやめるTrue Route
True Route第4章変形「お前、結局自分のことは何も言わないよな」溜めた3回分が一撃になる

運用上の注意​

  • 伏線は台詞で説明し直さない
  • 回収は、同じ言葉をなぞるだけでなく、意味が変わったことを場面で見せる
  • 日常モチーフは1回で終わらせず、少なくとも破綻前か終幕で再登場させる
