ルート概要

目的​

本資料は、共通ルートを抜けた後の各ルートの流れを、企画・実装・レビューに使える粒度でまとめる。 詳細なシーン構成は ストーリートリートメント と シーン分解表 を基準とし、本資料は各ルートの要点把握用とする。

Aルート 綾瀬川咲夜​

概要​

咲夜ルートは、表現が誰かを傷つけるかもしれないという恐れを扱う。 テーマは「罪悪感と告白」、結末は「自分の声を自分の責任で他人へ渡す」ことに置く。

感動の核​

  • 咲夜が「正しい言葉」ではなく、「自分の責任で渡す言葉」を選ぶ瞬間を頂点に置く
  • 泣かせどころは、謝罪が通るかどうかではなく、もう一度声を出す決断そのものに置く
  • 朔は救済者にならず、咲夜が自分で立つための余白を守る

前半​

  • 咲夜は出演辞退を申し出る
  • 朔は引き止めず、辞退理由を聞くことも急がない
  • 旧音楽室での会話を重ねる中で、中学時代の出来事が明かされる
  • 旧音楽室が、家録音を失ったあとの仮録音の避難場所でもあったと分かる
  • 朔は「歌えば救われる」と言わず、歌が暴力になり得ることを一度受け止める

中盤​

  • 咲夜は原稿や歌詞を破棄しようとする
  • 白瀬凪へ送れなかった言葉が、謝罪と表現責任の問題を具体化する
  • 家族に匿名活動が見つかったあと、家で声を出せなくなった経緯も重なり、声の出口そのものを失っていたと分かる
  • 凪のノートに残った 咲夜 の書き込みを見て、朔は初めてその読みを確かめる
  • 朔は止める代わりに、「これは誰の言葉として残したいのか」と問い返す
  • 咲夜は初めて、自分が誰かのためではなく、自分のために歌いたい気持ちを認める

序盤の仕込みと終盤の回収​

  • 仕込み:
    • 「誰にも見せたくない言葉」
    • 「見せる形にすると違うものになる」
    • 旧音楽室でだけ少し呼吸が整うこと
    • 旧音楽室が 歌う場所 ではなく 録れる場所 として機能していること
    • 咲夜 の読みがしばらく宙づりになること
  • 回収:
    • 舞台は隠れる場所ではなく、自分の責任で立つ場所へ変わる
    • 見せることへの恐れが消えるのではなく、それでも渡すと決める
    • 本番後、朔が初めて 咲夜 を さや と正しく呼ぶ
    • エンディングでは、朝の旧音楽室で隠れずに声を出す

後半​

  • 本番直前、咲夜は再び逃げかける
  • 七瀬の短い一言が、急がせないという方針を補強する
  • 朔は背中を押さず、選択を咲夜自身へ返す
  • 咲夜は短い導入のあと、自分の言葉として歌う
  • 間違った押し方をした場合は、舞台袖から出られない BAD END に落ちる

着地​

  • 過去の傷は消えない
  • だが、声は他人を傷つけるだけのものではないと知る
  • エンディングは朝の旧音楽室で、もう隠れずに声を出す場面で締める
  • 余韻は「赦された」よりも、「引き受けて生きる」に置く

Bルート 四辻千紘​

概要​

千紘ルートは、怒りと依存拒否を扱う。 テーマは「助けを拒むことでしか立てなかった人が、対等に支えられることを受け入れる」ことにある。

感動の核​

  • 千紘が怒りではなく、疲労と限界を自分の言葉で認める瞬間を頂点に置く
  • 泣かせどころは、壊れる場面ではなく、誰かに持たせることを初めて許す場面に置く
  • 朔は全部を引き受ける約束をせず、背負える範囲と背負えない範囲を言葉にする

前半​

  • 千紘は企画そのものを中止にしようとする
  • 朔は感情論で止めず、具体的な懸念を拾う
  • 家庭での負荷や、頼ることへの嫌悪が徐々に見えてくる
  • 朔は一度、優しさだけで近づいて失敗する

中盤​

  • 千紘は「どうせ誰も責任を取らない」と本音を漏らす
  • 水城先生が、曖昧な優しさでは届かないと釘を刺す
  • 朔は否定せず、自分が引き受ける範囲と引き受けられない範囲を明言する
  • 千紘は初めて、完璧に支えられる側になる必要はないと知る

序盤の仕込みと終盤の回収​

  • 仕込み:
    • 形にできないものへの拒絶
    • 電話口ににじむ疲労
    • 一人で片づけた方が早いという口癖
  • 回収:
    • 真帆や朔が持てるものを具体的に分ける
    • 千紘は「助けて」より先に「これを任せる」と言えるようになる
    • 文化祭翌日の昼に、肩の力を抜いた沈黙が初めて成立する

後半​

  • 本番前、家庭側のトラブルが重なり千紘は完全撤退を選ぼうとする
  • 真帆の連絡で、誰かが代わりに持てるものがあると示す
  • 朔は同情ではなく、選択を一緒に背負う姿勢を示す
  • 千紘は怒りではない言葉で自分の立場を語る
  • 企画優先や責任の押し戻しをすると、関係断絶の BAD END に落ちる

着地​

  • 千紘は弱さを見せても尊厳は失われないと知る
  • エンディングは文化祭翌日の昼、初めて肩の力を抜いて話す場面で締める
  • 余韻は「救われた」よりも、「一人で立ち続けなくていい」に置く

Cルート 穂坂結月​

概要​

結月ルートは、諦めと停滞を扱う。 テーマは「失ったものが戻らなくても、未来に触れる意思を持てるか」にある。

感動の核​

  • 結月が大きな夢ではなく、小さな一文を自分から選ぶ瞬間を頂点に置く
  • 泣かせどころは、劇的な復活ではなく、閉じていた未来を少しだけ開けることに置く
  • 朔は前向きさを要求せず、「今ここで選ぶ一歩」の意味だけを支える

前半​

  • 結月は企画から静かに距離を取ろうとする
  • 朔は励ましで押さず、結月の生活や家庭の現実を知っていく
  • 結月は「どうせ続かないから」と期待しない姿勢を崩さない

中盤​

  • 朔の善意は一度空回りし、結月を余計に疲れさせる
  • それでも朔が離れず、何も決めさせずに同じ場所へ来ることで、結月の沈黙が少し緩む
  • 家の前での会話と母の気配により、停滞が現在進行形の事情だと明かされる
  • 結月は失った機会の話を初めて自分の言葉で語る

序盤の仕込みと終盤の回収​

  • 仕込み:
    • 進路の話題から自然に降りる
    • 「終わりが決まってる作業は少し楽」
    • 欲しいものを口にしない癖
  • 回収:
    • 本番で読む一文が、結月にとって初めての現在形の意思表示になる
    • 書類や進路の話を自分から出す
    • 夕方から夜へ変わる通学路で、次の行き先を初めて口にする

後半​

  • 本番参加か辞退かで最後まで揺れる
  • クライマックスは、結月が自分の言葉を書いて、自分の声で読むこと
  • 他人の投書ではなく、結月自身が初めて「言えなかったこと」を文字にする行為が頂点
  • 四行の内容は大きな決意ではなく、「欲しかったもの」「行きたかった場所」「言いたかったこと」のような、結月が口にすることを自分に禁じてきた小さな望み
  • 書く過程で何度も消す。一行目が出てこない。外面的には地味だが、結月の内面では最も激しい戦いが起きている
  • 紙コップの裏に書く: 投書用紙に書く＝「正式に提出する」ことを意味し、結月にはそのハードルが越えられない。だが紙コップの裏なら、いつでも捨てられる。「本気じゃないふり」ができるから書ける。結月の「欲しいと言えない」構造が媒体の選択にまで貫通している
  • 結月が四行を読み終えた時、声は震えているが止まらない。それが「止まり続けることをやめた」証拠になる
  • 朔は未来を約束しないまま、今ここで選ぶ意味だけを示す
  • 押しつけるか見放すと、結月が静かに輪から外れる BAD END に落ちる

各ルートの文化祭本番​

文化祭本番の見え方は、ルートによって異なる。

ルート本番の核設計意図
A咲夜が朗読の途中で歌に変わる瞬間がある。台本にない、咲夜自身の声「声を出す」決断が行動として可視化される
B千紘が裏方に徹するが、本番中に一度だけ客席から企画を見る管理者ではなく観客として初めて作品を受け取る
C結月の四行が読まれる。結月自身が読むか代読かで最後まで揺れ、自分で読む止まっていた人間が動き出す行為そのもの

着地​

  • 結月は希望を持ち直すというより、未来を完全には閉じないことを選ぶ
  • エンディングは夕方から夜へ変わる通学路で、結月が初めて自分から行き先を口にする場面で締める
  • 余韻は「明るくなった」ではなく、「止まり続けることだけはやめた」に置く

隠し派生​

  • 四行目のあと の後段として、あかりの hidden epilogue を接続する
  • 役割は「結月が少し動き出したあと、支えていたあかり自身の止まっていた部分を見る」こと
  • 感動の核は、結月が進めた嬉しさと、自分の役目が空いた喪失が同時に来る痛み
  • 余韻は恋愛逆転ではなく、「結月の友達」ではないあかり個人へ視線が移ることに置く

あかりEND 柊あかり​

概要​

あかりENDは、舞台の外から場を支えていた相手へ視線が向く終わり方である。 テーマは「支える側もまた見られたい」、結末は「誰かの友達としてではなく、あかり本人を見る」に置く。

感動の核​

  • あかりの明るさが単なる賑やかしではなく、場と人間関係を繋ぐための実務と気遣いだったと分かる瞬間を核に置く
  • 泣かせどころは告白そのものではなく、「外側にいた人が、初めて自分へ向いた視線を受け取る」ことに置く
  • 朔はあかりを結月の補助線として消費せず、本人の役割と痛みを個別に受け取る

流れ​

  • 第3章で、あかりは結月に対する優しさの危うさを朔へ指摘する
  • 第4章後、誰の個別ルートにも届かなかった時に、あかりが企画の立て直しを提案する
  • あかりは広報・導線・客入れを引き受け、表に立つ五人を外側から支える
  • 本番前後で、結月が昔の癖で「あかちゃん」と呼び、止まっていた中学時代の距離が一瞬だけ戻る
  • エンディングでは夜道で、朔が「結月の友達」ではないあかり自身を見ると約束する

着地​

  • あかりは「見てもらう」ことと「消費される」ことの違いを知る
  • 朔は舞台の中心だけでなく、それを成立させていた外側の人間も見ていたと示す
  • 余韻は「恋が始まった」より、「この先をちゃんと見てみたい」に置く

将来の派生案​

  • 結月ルート後の隠しあかりENDでは、fallback版より一段だけ静かな距離で 支える側の余白 を描く
  • ここではあかりを攻略対象化するというより、結月ルートで見えていた関係の裏面を返す

いろは hidden 候補​

概要​

いろは hidden は、咲夜 hidden のさらに後段で開く補助的エピローグとして扱う。 テーマは「前に出ないことで守っていた距離が、信頼に変わるか」に置く。

方向性​

  • 咲夜ルートなら 声を届ける光 の延長で、光を当てる側の覚悟へ踏み込む
  • 中学時代に一度だけ聞いた学校での歌と、夜に聞いていた SakuYa 名義の匿名音源が静かに繋がる構造が使える
  • SakuYa は S と Y だけを不自然に立てた中学生っぽい名義で、後から見ると咲夜本人の半端な隠し方だと分かる
  • 本編ヒロインの感情決着を壊さない後日談構造に限定する
  • 感動の方向は 見守っていた側もまた、見てほしかった に寄せる
  • 本編中は 昔に聞いた声が残っている までに留め、救われていた の告白は hidden 側に回した方が伸びる

友情END​

概要​

友情ENDは、誰か一人の夜を選ばず、それでも全員の夜を見なかったことにはしなかった場合の終わりである。 テーマは「選ばないことと見捨てないことの両立」にある。

着地​

  • 企画は縮小版として立て直される
  • 三人はそれぞれの傷を抱えたままでも、同じ場に立てるところまでは戻る
  • 余韻は恋愛成就ではなく、「また明日、全員に会える」の重さに置く

共通BAD​

概要​

主人公が企画の体裁や正しさを優先し、誰の夜にも本気で踏み込まなかった場合の終わり。

結果​

  • 企画は中止
  • 3人はそれぞれ元の距離へ戻る
  • 朔はまた「聞ける時に聞かなかった」ことを繰り返したと知る
