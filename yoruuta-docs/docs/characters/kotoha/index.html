椎葉 殊羽（しいば ことは）

基本情報​

  • 名前: 椎葉 殊羽（しいば ことは）
  • 学年: 高校1年生
  • 立場: 図書委員の後輩。文化祭では投書箱の回収と管理を手伝う

役割​

  • 元気で可愛い方言キャラクター: この海沿いの町で生まれ育った地元っ子として、舞台の空気を声と笑顔に載せる
  • いろはとは別系統の後輩ポジション。いろはが清楚・裏方なら、殊羽は愛嬌・窓口
  • あかりに近いエネルギーを持つが、あかりが「華やかさ」なら殊羽は「素朴な可愛さ」。ギャルと地元っ子の差
  • 投書箱に言葉を預ける側の視点を持つ（いろはは届ける側、殊羽は預かる側）
  • 博多弁の柔らかさと元気な声が、主要キャラの標準語の緊張感と対比を作る
  • 笑顔とテンションで場を和ませる存在だが、投書を預かる場面では急に真剣になる。この落差が殊羽の物語上の強み
  • 天然・いじられ枠。抜けているところが多く、あかり・湊・いろはに日常的にいじられる。殊羽がいる場面は空気が軽くなる

いろは・あかりとの差別化​

項目いろは殊羽あかり
系統清楚・裏方愛嬌・窓口華やか・盛り上げ
口調敬語（素が出ると崩れる）博多弁（緊張すると標準語に寄る）砕けた口語（じゃん・でしょ）
場への関わり機材・照明で場を支える投書の回収・来場者の誘導で場を繋ぐ掲示・広報で場を盛り上げる
エネルギー低〜中。控えめに見守る高。自分から距離を詰める高。場の中心に立つ
可愛さの質気遣いの健気さ笑顔と愛嬌の天然物華やかさとギャップ
性格空気を読みすぎて間合いを測る天然・猪突猛進。抜けてていじられる軽そうに見えて観察が速い
いじられ度ほぼなし（後輩の礼儀がある）高。あかり・湊・いろはの標的自分がいじる側
ヒロインとの接点咲夜（憧れ）結月（図書委員の先輩）結月（中学の友人）

性格​

  • 笑顔が武器。本人に自覚はないが、笑うと周りの空気が軽くなる。それが天然の愛嬌
  • 元気で人懐っこく、初対面でも自分から距離を詰める。馴れ馴れしいのではなく、壁を作らないだけ
  • 感情がすぐ顔に出る。嬉しい時は全力で嬉しいし、怒る時は全力で怒る。隠せない
  • 声が大きめ。図書室で注意されたことがある
  • 猪突猛進。思い立ったら即行動。確認する前に走り出す。結果として失敗やドジが多いが、本人はあまり反省しない
  • 天然で抜けている。話を聞いているようで肝心なところを聞き逃す。伝言を途中まで忘れる。左右を間違える。日付を一日ずれて覚える
  • あかり・湊・いろはにとって格好のいじり対象。本人はいじられていることに気づいていないか、気づいても「えー、なんでー？」で流す。それがまた周囲を笑わせる
  • テンションの高さはあかりに近いが、あかりが場を「回す」知性を持っているのに対し、殊羽は場に「飛び込んで転ぶ」タイプ。あかりが突っ込み、殊羽がボケの構図になりやすい
  • 言葉の裏を読むのは苦手。冗談や皮肉をそのまま受け取って真顔で返すことがある。ただし感覚的に人の温度を測れるので、悪意には敏感
  • 「投書に何を書いたか」は絶対に聞かない。預かった言葉を守る本能がある。普段のテンションとのギャップがここに出る
  • 緊張すると方言が薄くなり、標準語に寄る。本人はこれが恥ずかしい
  • ぶりっ子ではない。可愛く見せようとしていない。素で可愛いのであって、計算がない。むしろ本人は「かわいい」と言われると照れて否定する

方言設計​

方針​

  • 博多弁をベースにする。この町の方言が博多弁圏に近いという扱い
  • 現代の福岡の女子高校生が実際に使う表現を基準にする。メディアが作った「ステレオタイプ博多弁」は避ける
  • 主力語尾は 〜っちゃん 〜と？ 〜けん。これが殊羽の方言の三本柱
  • コテコテにしすぎない。語尾と感嘆詞に集中させ、文の骨格は標準語に近い
  • 方言は「この子が地元の子だ」と分かる程度。全文方言にはしない

使用する博多弁要素（現代の若者が実際に使うもの）​

要素用例効果使用頻度
〜っちゃん / 〜っちゃんね「そうっちゃん」「買ったっちゃん！」断定・報告の語尾。現代博多弁の主力。可愛さが自然に出る高
〜っちゃけど「聞きたいっちゃけど」「今日バイトっちゃけど」「〜なんだけど」。柔らかい前置き高
〜と？「何しとーと？」「行くと？」疑問の語尾。博多弁で最も可愛さが出る要素。老若男女で現役高
〜けん / 〜やけん「寒いけん、閉めるね」「うちがやるけん」理由の接続。世代問わず自然高
ばり「ばり好き」「ばりうまい」「とても」の意。若者に最も定着した博多弁副詞中〜高
あーね「あーね、分かった」相槌。福岡の若者のほぼ全員が使う中
〜しとう / 〜しとる「知っとう？」「まだ残っとうよ」進行形。日常使用は現役中
なんしようと？「なんしようと、こんな時間に」「何してるの？」。丸みのある問いかけ中
よか「よかよか」「それでよかと？」肯定・許容。温かみがある低〜中

避ける・控える表現（年配層・ステレオタイプ寄り）​

要素理由
〜たい（語尾）若者は 〜っちゃん に移行済み。使うとしてもごく稀。殊羽のおばあちゃんが使っていて、つい出る程度
〜ばい同上。衰退が顕著。若い女性はほぼ不使用
ばってん絶滅危惧。若者・女性はほぼ使わない
〜くさ年配男性限定。女子高校生が使うと不自然
好いとう（告白用法）メディアが作った博多弁の象徴だが、実際の福岡女子は「好きっちゃん」と言う

方言の制御​

  • 先輩に対しては博多弁+敬語の混合（「〜しとるんですけど」「よかったですか？」）
  • 同級生（いろは等）には素の博多弁が出やすい（〜っちゃん 〜と？ が増える）
  • 緊張すると方言が引っ込む。これが鎧が外れる構造の逆（素の方がリラックスしている証拠）
  • あかりの関東弁「じゃん」と殊羽の「〜っちゃん」「〜と？」が並ぶと、町の内と外が自然に見える
  • 〜たい 〜ばい は「おばあちゃんの口癖がうつった」扱いで、ごく稀に出る程度。殊羽の家庭環境（祖父母と近い距離で育った地元っ子）を匂わせる味付けに使う

背景​

  • この町で生まれ育った。海の近くの住宅地に住んでいる
  • 父は漁協関連の仕事、母はパート。普通の家庭
  • 図書委員になったのは「静かな場所が好き」ではなく「本を読むのが好き」だから
  • 結月とは図書委員の先輩後輩として面識がある。結月が本の話だけは少し饒舌になることを知っている
  • 投書箱の管理を引き受けたのは、図書委員で「匿名の言葉を扱う」経験（貸出カードや推薦カード）があったから
  • あかりとは文化祭の準備で接点ができ、すぐに懐いた。あかりのおしゃれや立ち居振る舞いに素直に影響を受けている
  • インナーカラーの青メッシュは、あかりに「ことはちゃんも何か入れてみなよ」と勧められて入れた。海っぽい色を選んだのは殊羽のセンス

見た目​

  • 身長155cm前後
  • 黒髪のセミロング、毛先が少し外にはねる。動くたびに揺れる軽さがある
  • インナーカラーに青メッシュを入れている。あかりの影響でおしゃれに目覚めた証。耳にかけた時や風で髪が揺れた時にちらっと見える
  • 前髪は目にかからない程度に切り揃えている。表情がよく見える
  • 日焼けが少しだけ残っている（海沿いの生活感）
  • 笑うと目が細くなり、頬が上がる。この笑顔が殊羽の最大の個性
  • 制服はきちんと着ているが、カーディガンの袖を少し余らせている
  • 表情が豊かで、感情が顔に出やすい。喜怒哀楽のリアクションが大きい
  • 配色はアイボリー、薄い水色、砂色

デザイン意図​

  • いろはの「清楚な後輩」と並んだ時に、「元気で可愛い後輩」として一目で区別できる
  • あかりの「華やかさ」とも被らない、素朴だが愛嬌のある印象。ただしメッシュが「おしゃれに気を使い始めた後輩」の成長感を出す
  • 海と日差しを連想させる配色で、舞台の空気感を視覚的に持たせる。青メッシュは海の色と繋がり、殊羽の地元っ子感を強化する
  • 立ち絵の基本表情は笑顔寄り。他のキャラが真顔やクール系の基本表情を持つ中で、殊羽だけが「笑っている」ことで差別化する

うち（博多弁として自然。ただし先輩の前では「私」に寄ることもある）

口調​

  • 博多弁ベースだが、重くない
  • テンポは速め。思ったことがすぐ口に出る
  • 感情が動くと声が高くなり、語尾が跳ねる
  • リアクションが大きく、驚きや喜びが声量に直結する
  • 先輩には敬語と博多弁の混合だが、テンションが上がると敬語が飛ぶ

物語上の機能​

投書箱の番人​

  • 投書箱の回収と管理を担当する。誰が何を書いたか知り得る立場にいるが、絶対に言わない
  • 「預かった言葉」への誠実さが、企画の信頼を支えている
  • 第4章の漏洩時、殊羽が管理を任された投書が軽く消費されることで、殊羽自身も傷つく
  • ただし殊羽の傷は物語の中心にはならない。あくまで「投書を預けた人たちの信頼が壊れた」ことの触媒

結月との接点​

  • 図書委員の先輩後輩として、結月と静かな時間を共有している
  • 結月が本の話だけ少し饒舌になることを知っている唯一の後輩
  • Cルートで結月が四行を書く時、投書箱にそっと入れたのを殊羽が受け取る可能性がある
  • ただし結月の決断を代弁しない。受け取るだけ

いじられキャラとしての機能​

  • 殊羽の天然とドジは、場の緊張を自然に解く装置として機能する
  • あかり: ツッコミ担当。殊羽のボケに最速で反応する。「ことはちゃんさあ……」が定型
  • 湊: 軽口でいじる。殊羽が真に受けて慌てるのを楽しんでいるが、悪意はない
  • いろは: 同学年なので遠慮なくいじれる。普段は気遣いのいろはが殊羽相手だと素が出やすい。「ことはちゃん、それ二回目だよ」と冷静にカウントする
  • 殊羽はいじられても根に持たない。むしろ周りが笑ってくれること自体を無意識に嬉しがっている
  • 重要: いじりは「殊羽が傷つかない範囲」に必ず留める。天然やドジをいじるのであって、方言や出自をいじるのは禁止

あかりへのリスペクトと対比​

  • いろはが咲夜を尊敬するように、殊羽はあかりをリスペクトしている
  • あかりの華やかさ、おしゃれのセンス、場を仕切る力に素直に憧れている
  • 青メッシュはあかりの影響。「柊先輩みたいにはなれんけど、うちもちょっとだけ」という距離感
  • ただし殊羽のリスペクトは「なりたい」ではなく「すごかー！」という純粋な賞賛。自分を変えたいというより、あかりの隣にいて楽しい
  • あかりは関東から来た転校生。殊羽は地元育ち。あかりの「じゃん」と殊羽の「〜と？」が並ぶと、この町の内と外が自然に見える
  • 二人ともエネルギーが高いが質が違う。あかりは「場を回す」知性がある。殊羽は「場に飛び込む」勢いがある
  • 二人が会話するとテンポが一気に上がり、場が最も明るくなる。コメディの最強ペアとして使える
  • あかりは殊羽の天然の可愛さを面白がる側（「ことはちゃんそれ自分で分かってないの最強じゃん」的な反応）

  • あかりとの会話は「じゃん」と「〜と？」の対比を1回だけ見せれば十分。繰り返しすぎない
  • 投書に関わる場面ではテンポが落ち、方言も少し抑えめになる（預かった言葉への真剣さ）。普段の元気さとのギャップが殊羽の深みになる
  • 殊羽の方言が可愛く見えるのは「狙っていない」から。自覚させない
  • 〜と？ の疑問形と 〜っちゃん の断定形が殊羽の方言の二大柱。この二つで博多弁らしさの8割を担う
  • 〜たい 〜ばい は基本使わない。使う場合は「おばあちゃんの口癖がうつった」演出として、ごく稀に1回だけ出す程度
  • 愛嬌の表現: ！ 多め、リアクション大きめ、声量の変化で元気さを出す。ただし全台詞が ！ で終わるとうるさいので、静かな場面との落差を意識する
  • 可愛さの扱い: 殊羽が可愛いことは他キャラの反応（「お前元気すぎ」「笑うなよ、つられるから」等）で間接的に見せる。本人に「可愛い自分」の自覚はない
  • 天然・いじりの運用: 殊羽のドジや勘違いは1シーンに1〜2回で十分。やりすぎると「ただのアホ」になる。投書を預かる場面の真剣さとの落差があるからこそ天然が映える
  • いじる側の配分: あかりが最も頻度高くいじり、湊が軽口で乗り、いろはが冷静にツッコむ。3人の反応の違いで殊羽への距離感が見える

名前の採用理由​

  • 椎葉 は宮崎に集中する九州特有の姓。全国的には珍しく、読みの「しいば」がs音で始まり柔らかい
  • 殊羽 は「特別な羽」。珍しい漢字の組み合わせだが読みは「ことは」で親しみやすい。漢字の硬さと博多弁の柔らかさにギャップが出る
  • いろは（平仮名）・あかり（平仮名）と並んだ時に、漢字二文字で視覚的に差別化できる
  • 「ことはちゃん」と呼ばれた時の四音の響きが自然で、後輩ポジションに馴染む
