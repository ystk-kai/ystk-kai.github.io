ビジュアル仕様書

目的​

本資料は、主要キャラクターの見た目をラフ制作へ渡せる粒度まで具体化する。 物語上の役割説明ではなく、立ち絵、イベントCG、キービジュアルでぶれないための視覚仕様を定義する。

全体方針​

  • 今どきの恋愛ADVらしい華やかさは保つ
  • ただし、流行優先で全員を似た顔にしない
  • 立ち絵で最初に識別される要素を、髪 / 輪郭 / 姿勢 / 色 の4点で分ける
  • 顔立ちは端正寄りに統一しつつ、目の形と重心で個性を出す
  • バスト差や体格差は入れるが、記号化しすぎない

画面に並べた時の役割​

キャラクター第一印象シルエット画面上の役割
咲夜静かな主役縦長最も視線を止める
千紘緊張と意志横広がり画面を締める
結月柔らかい余白曲線画面を呼吸させる
いろは後輩の機動力小柄で軽い画面のリズムを変える
あかり派手な外圧と親しさ崩した制服と装飾画面に雑談の熱と色を足す
朔中立情報量少なめヒロインを受ける

主人公: 久瀬 朔​

顔立ち​

  • 輪郭は細めだが尖らせすぎない
  • 目はやや切れ長で、無表情だと冷たく見えるが、造り自体は端正
  • 灰がかった青の瞳。光の当たり方で印象が変わり、近くで見た時に記憶に残る
  • 眉は整っているが主張を強くしない
  • まつ毛表現は最小限

髪​

  • ダークブラウン。光の加減で茶色みが分かる程度
  • 前髪は軽く流す
  • 束感は控えめ

体格​

  • 176cm前後
  • 細身だが弱く見えない
  • 肩幅は標準より少し狭い

立ち姿​

  • 重心は中央
  • ポケットへ手を入れない
  • 説明的なポーズを避ける

魅力の方針​

  • 派手な美形ではなく、表情・声・佇まいで引き込む「雰囲気イケメン」路線
  • 本人に外見への自覚がない。湊に「怖い顔」と言われても「普通だろ」と返す程度
  • 見た目単体ではなく、聞く姿勢や声の質感と合わせて初めて印象に残る設計にする
  • うたシリーズ主人公（深桜・新海）の系譜。華やかさより空気感で惹きつける

ヒロイン​

  • 咲夜
  • 千紘
  • 結月

綾瀬川 咲夜​

顔立ち​

  • 面長寄りの小顔
  • 目は細めだが、つり目にはしない
  • 上まぶたの線を繊細にして、伏し目が映える形にする
  • 唇は薄め

前髪と髪型​

  • 薄めの前髪
  • 目に少しかかる長さ
  • 胸下から腰手前までのストレートロング
  • 顔まわりに細い後れ毛だけ残す

体格​

  • 168cm前後
  • スレンダー
  • バストは小さめから標準未満
  • 肩幅は狭い

立ち姿​

  • 体の正面をあまり見せない
  • 肩を少し内側へ入れる
  • 腕は身体に近い位置で収める

表情の軸​

  • 通常: 無表情ではなく、言いかけで止まっている顔
  • 微笑: 口角だけ小さく上がる
  • 困り: 眉だけ少し寄る
  • 泣き: 大きく崩さず、目元にだけ出す
避けること
  • 病弱系の記号に寄せない
  • 前髪を重くしすぎない
  • 胸や脚で売る方向にしない

四辻 千紘​

顔立ち​

  • 輪郭はややシャープ
  • 目は横幅広めの切れ長
  • 眉は直線寄り
  • 鼻筋と首筋が通って見える描き方にする

前髪と髪型​

  • 分け目は浅め
  • 顎下ボブ
  • 毛先はそろえて切り揃える
  • 外はねやふわ感は抑える

体格​

  • 157cm前後
  • 3人の中で最も小柄
  • 細身だが骨格はしっかり
  • バストは標準

立ち姿​

  • 肩線を水平に近く保つ
  • 顎を引き、目線の鋭さと姿勢の正しさで圧を出す
  • 腕組みより、片手に資料やペンを持たせる方がらしい

表情の軸​

  • 通常: 不機嫌ではなく、判断中の顔
  • 微笑: かなり希少。口元だけで作る
  • 怒り: 目を吊るより、口調と眉で見せる
  • 崩れ: 目線が落ちた時の差を大きくする
避けること
  • ツンデレ記号に寄せない
  • ボブを可愛く丸めすぎない
  • 露出や派手アクセで強さを出さない

穂坂 結月​

顔立ち​

  • 輪郭は丸すぎない卵型
  • 目はやや丸みがあるが、幼くしすぎない
  • 下まぶたに薄く柔らかい線を入れる
  • 笑顔でも目を大きく開きすぎない

前髪と髪型​

  • 前髪は薄く流し、額を少し見せる
  • 肩甲骨下までのゆるいロング
  • 低めのひとつ結び、または片寄せでまとめる
  • 結び跡のゆるさを残す

体格​

  • 160cm前後
  • やや細身
  • バストは標準よりやや大きめ
  • 立つと少し頼りなく見える重心にする

立ち姿​

  • 重心を少し後ろへ逃がす
  • 膝とつま先をまっすぐ揃えすぎない
  • 手先の力を抜く

表情の軸​

  • 通常: 柔らかい微笑の手前
  • 微笑: 安心より、相手を気遣う笑い
  • 困り: 苦笑に寄せる
  • 本音: 目線が逃げなくなった時に一番映える
避けること
  • ただのおっとりキャラにしない
  • 幼く見える丸顔に寄せない
  • ロングの量感を重くしすぎて、咲夜と近づけない

サブキャラクター​

  • いろは
  • あかり

御子柴 いろは​

顔立ち​

  • 小柄だが、子供顔にはしない
  • 目は素直な形で、観察の速さが出る
  • 口元は締まり気味

前髪と髪型​

  • 肩下ロング
  • 低めにひとつへまとめることが多い
  • 仕事中に崩れにくい実用寄り

体格​

  • 150cm前後
  • 細く軽い
  • バストは控えめ

立ち姿​

  • 少し前傾
  • 先輩相手では一歩引く
  • 物を持っている姿が似合う

柊 あかり​

顔立ち​

  • 目は大きめだが、丸く幼くはしない
  • まつ毛とアイラインで華やかさを出す
  • 口元は笑うと強いが、真顔になると観察の速さが出る形にする
  • 盛れている顔 だが、作り物っぽくしすぎない

前髪と髪型​

  • ベースは毛先を巻いたロングヘア
  • 髪色は明るいシャンパンゴールド
  • メッシュは白寄りのアッシュシルバーを主役にする
  • ピンクベージュは差し色として少量だけ混ぜる
  • 前髪の内側、顔まわり、表面の数束にアッシュシルバーを入れ、立ち絵で一目で分かるようにする
  • ピンクベージュは内側と毛先寄りに散らし、近くで見た時に気づく情報量として使う
  • 巻きは強すぎず、毛先を中心に遊ばせる
  • 結んでも下ろしてもギャル感が残るよう、前髪と顔まわりの情報量を多めにする

服装​

  • ブレザーは着ない
  • 学校指定のワイシャツは第二ボタンまで開ける
  • リボンはきっちり結ばず、ゆるく落として着崩す
  • 上から大きめのカーディガンを羽織り、袖は肘近くまで無造作にまくる
  • スカート丈は校則ギリギリの短め
  • 足元はルーズソックスで、全体の重心を下に落とす

小物​

  • 手首に細いブレスレットやヘアゴム、チャームを複数つける
  • 指輪は多用せず、手首まわりのじゃらつきでキャラを出す
  • ネイルは長すぎないショート寄りだが、ラメやミルキーカラーで存在感を出す
  • スマートフォンケースやポーチにも軽いデコ感を入れてよい

体格​

  • 162cm前後
  • 細身だが華奢にはしない
  • 姿勢と服の崩し方で存在感を作る
  • バストや脚の強調より、全身の情報量で目を引かせる

立ち姿​

  • 片脚に重心を逃がす立ち方が似合う
  • 腕まくりした手元や、カーディガンの袖を引っかける仕草が映える
  • 笑っている時は前のめり、真面目な時は急に静止する差を大きくする

表情の軸​

  • 通常: 人と話す準備ができている明るい顔
  • 笑い: 口が先に動く、反応の速い笑い
  • いら立ち: 眉を少し上げて言い返す、怒鳴らない
  • 本気: 目元の力だけを残して口元の笑いが消える
避けること
  • 不良や威圧系に寄せない
  • メイクを濃くしすぎて年齢を上げすぎない
  • 露出だけで派手さを作らない
  • ギャルなのに実は清楚 のような安いギャップにしない

比較メモ​

  • 咲夜と結月はどちらもロングだが、咲夜は 直線と静止、結月は 曲線と抜け で分ける
  • 千紘は唯一の短髪で、一覧とキービジュアルの支点になる
  • いろはは小柄ロングだが、主役ヒロインほど髪量を持たせない
  • あかりは髪色、制服の崩し、小物量で一目で識別できるようにする

ラフ制作の優先順​

  1. 咲夜、千紘、結月の立ち姿シルエット比較
  2. 3人の顔アップ比較
  3. いろは、あかりのサブ立ち絵要否判断
  4. 朔の簡易立ち絵判断
