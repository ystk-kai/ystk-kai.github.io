四辻 千紘（よつつじ ちひろ）

基本情報​

  • 名前: 四辻 千紘（よつつじ ちひろ）
  • 学年: 高校3年生
  • 立場: 文化祭実行委員の中心、学内では厳しい管理役

物語上の役割​

  • 緊張と衝突を作る
  • 言葉が人を傷つける側面を見せる

感情の核​

  • 怒り
  • 反発
  • 弱さを見せることへの拒絶

家庭環境​

四辻家​

  • 父: 四辻 恒一: 以前は建設会社に勤めていたが、転職を繰り返し現在は不安定な就労状態。人当たりは悪くなく、悪意もない。だが継続的な責任を引き受けられない。「今度こそ」と言っては続かない
  • 母: 千紘が小学校高学年の時に家を出た。離婚が成立している。千紘は母を恨んでいるわけではないが、「結局いなくなる人」として処理している
  • 弟（名前未定）: 中学生。塾に通っている。家庭の事情をうっすら分かっているが、姉が何をしているかの全体像は見えていない

家庭が千紘に与えた影響​

  • 母の離婚が原点: 母が出ていった後、家事・弟の世話・家計管理が千紘に落ちてきた。誰に頼まれたわけでもなく、やらなければ家が回らなかったから自分でやった
  • 父の「ごめん」パターン: 恒一は反省する。だが同じことを繰り返す。千紘にとって「謝る」は意味のない行為になった。だから千紘は謝罪より行動を見る
  • 弟の存在: 千紘が家を投げ出さない最大の理由。弟が困らないように塾の振込をし、晩飯を作り、学校の提出物を確認する。千紘の「管理能力」は学校で身についたものではなく、家庭で生き延びるために覚えた生存技術
  • 経済的には崩壊していない: 父の収入と養育費で最低限は回る。だが余裕はなく、千紘が家計簿をつけて管理している。紙コップの会計にこだわるのは、日常の中で「数字を管理する」ことが千紘の安心装置だから
  • 家に「帰りたくない」わけではない: 弟がいるから帰る。だが家は休める場所ではなく、もう一つの仕事場

咲夜との対比​

  • 咲夜: 家は広い。経済的に困らない。だが声を出せない（内的な檻）
  • 千紘: 家は狭くはないが余裕がない。だが休めない（構造的な檻）
  • 二人とも家庭に縛られているが、縛り方が全く違う

物語上の機能​

  • 「一人で片づけた方が早い」の根拠: 効率主義ではない。任せた相手が途中で投げ出す経験の蓄積から来ている
  • 朔への態度の背景: 善意だけで「手伝う」と言う人間を千紘が信用しないのは、父がまさにそのパターンだから
  • Bルートの核心: 千紘が朔に「任せる」と言えるようになることの重さ。それは父に裏切られ続けた信頼を、もう一度誰かに預けること
  • 第4章の電話: 朔の前で父からの電話に出ざるを得ない場面。千紘が管理している「表の顔」が崩れる瞬間
  • 明示しすぎない: 父との関係は千紘の行動（帰る時間を気にする、弟の話をする、家計を管理する）から推測させる。父との直接対話シーンは最小限に

抱えている夜​

家庭では長く不安定な父親の世話と生活のフォローを担ってきた。 母が家を出た後、謝るだけで変わらない父を見続けた結果、

  • 弱音を吐く人間は、誰かに支えてもらう前提でいる
  • 甘えることは、誰かに負担を押しつけることだ

という考えを強く持っている。

そのため千紘は、

  • 助けを求める人に厳しい
  • 自分が助けを受けることにはさらに厳しい

現在の状態​

  • 学校では有能で冷静な先輩に見える
  • 実際には、限界まで抱え込むことでしか立っていられない
  • この企画を、傷の見せ合いにしか見えず嫌悪している
  • 帰宅後は弟の夕飯を作り、家計簿をつけ、父の帰りを確認してから自分の勉強を始める

性格​

  • 反応が速い
  • 正論で話す
  • 観察力が高く、人の甘えにも鋭い
  • ただし本質は不誠実さへの恐れ

不器用な気遣い（ツンデレ的側面）​

千紘の厳しさは嗜虐ではなく、失敗を見たくないことから来ている。そのため、本人は認めないが行動で人を助けている場面が頻繁にある。

  • 「確認したついでに直しておいた」（わざわざ確認しに行っている）
  • 飲み物を買ってきた時は「自販機が二つ出しただけ」と言う
  • 朔の企画書の誤字を黙って直す。指摘はしない
  • 疲れている相手に「帰りなさい」と言うが、その前に作業の引き継ぎメモを書いている

この行動パターンをプレイヤーに言葉より先に見せることで、千紘の本質が管理者ではなく保護者だと分かる。

ドライユーモア​

千紘の観察力と断定口調は、批判だけでなくキレのある面白さにも使える。湊のような明るいツッコミではなく、状況を正確に言語化しただけの冷たい可笑しさ。

  • 怒りではなく呆れ寄りの場面で出す
  • 本人は面白いことを言ったつもりがない
  • 周囲が笑った時に「何がおかしいの」と本気で返す → ギャップが可愛さになる

鎧が外れる瞬間​

千紘を可愛いと思わせる設計上の重要ポイント。頻度は章あたり1回に絞る。多すぎるとツンデレテンプレになる。

  • 甘いものに弱い: 紙コップの砂糖を多めに入れる。指摘されると「効率の問題」と返す
  • 感謝に弱い: まっすぐ「ありがとう」と言われると対応が遅れる。一瞬黙ってから「当然のことをしただけ」
  • 名前を呼ばれた時の反応: 「四辻先輩」ではなく「千紘先輩」と呼ばれた瞬間、一拍遅れる
  • 深夜の連絡: 企画の相談で遅い時間に連絡すると、昼より少し声が柔らかい（本人は気づいていない）

見た目​

  • 身長157cm前後
  • 赤みを抑えたダークチェスナットの顎下ボブ
  • 毛先は内にも外にも逃がさず、切り揃えた硬さを残す
  • 瞳は琥珀寄りの濃茶
  • 制服はブレザーをきっちり着込み、腕まくりや着崩しはしない
  • シルエットは肩線がはっきり出るようにして、Aと真逆の横の強さを出す
  • 配色はボルドー、ダークブラウン、アンバー

デザイン意図​

  • 3ヒロインの中で最も輪郭が強く見えるデザインにする
  • 温度の高い色を使うが、可愛さより緊張感を優先する
  • 目つきだけでなく、姿勢と首筋の角度で厳しさを見せる
  • ロング主体になりやすい恋愛ADVの中で、一覧の引っかかりを作る短髪枠として機能させる

一人称​

私（正確さと距離感を優先する）

口調​

  • 断定が多い
  • 論点から話す
  • 曖昧な表現を嫌う
  • 感情が揺れた時ほど、語尾が短くなる

記号使用指針​

記号頻度方針
！やや少論理的な強調・苛立ちの表出に使う。叫びではなく切れ味
ー稀正確な話し方をするので音を伸ばさない
っ中歯切れの良さを出す。「はっきり」「やっぱり」など断定系と相性が良い
……やや少 → 本音で増普段は使わない。感情が漏れた時に初めて出現し、ギャップを生む
？多相手の曖昧さを追及する武器。千紘らしい切り込み

例: 「曖昧なまま進めるのが一番危ないの！」「本当に分かってる？ 体裁って具体的に何か言える？」

呼称変化のイベント設計（千紘ルート）​

  • 朔は基本的に 四辻先輩 を維持する
  • 終盤で 千紘先輩 に変える場合は、距離が縮まったことを示す効果がある
  • 千紘が 朔 と名で呼ぶのは、感情が完全に剥き出しになった場面に限定する

初登場​

  • 文化祭企画の打ち合わせ
  • 曖昧な企画案に対し、千紘が真正面から欠点を列挙する
  • 周囲は冷たいと感じるが、朔だけがその必死さに気づく

主人公との関係進展​

  • 面倒な理想論を持ち込む後輩
  • 感情で反論してこない相手
  • 弱さを見せても関係を切らない相手

台詞サンプル​

日常​

  • 「曖昧なまま進めるのが一番危ないの！」
  • 「感想じゃなくて、必要な作業を言って」
  • 「疲れてるなら休んだら。中途半端が一番迷惑よ」
  • 「空欄のまま出すくらいなら、まだ出さない方がまし」

反発​

  • 「優しさだけでどうにかなると思ってるなら甘いわよ」
  • 「あなたたちは、言えば誰かが受け止めてくれる前提で話してる」
  • 「弱いことと、弱さを振り回すことは違うでしょう！」

ドライユーモア​

  • 「十分で終わる仕事を三十分かけて丁寧にやった、って報告しないで」
  • 「予定通りに遅れてるなら、それは予定通りって言わないの」
  • 「今の、褒めてるつもり？ 傷つけてるのと紙一重だけど」
  • 「善意の残業は残業に含まれるの。帰って」

不器用な優しさが漏れる時​

  • 「……別に。確認のついでに見ただけ」（朔の企画書を直しておいて）
  • 「あなたの分じゃないわよ。自販機が二つ出しただけ」（飲み物を渡して）
  • 「帰りなさい。――これ、引き継ぎ用。明日やればいいから」
  • 「疲れてるなら休んだら。……椅子、もう一つ出すから」

鎧が外れる瞬間​

  • 「……っ。当然のことをしただけよ」（まっすぐ礼を言われて）
  • 「砂糖はエネルギーの問題でしょう。味の話はしてないわ」（紙コップの砂糖を指摘されて）
  • 「ち、千紘って……。急に呼ぶのやめてくれない？」
  • （作業中に寝落ちした朔にカーディガンを掛けて）「……寒いと効率が落ちるから」

本音が漏れる時​

  • 「頼るのが嫌なんじゃない。頼った先で崩れるのが嫌なの」
  • 「結局、最後に残るのは……自分で片づける側なのよ」
  • 「……放っておいてほしいわけじゃない。でも、うまく言えない」

終盤​

  • 「一緒に背負うなんて、簡単に言わないで！」
  • 「それでも……今だけは、ひとりで立てるふりをやめたい」

核心台詞候補​

  • 「弱いまま頼るのは負けじゃないって、今は少しだけ思いたい」

場面別比較: 相手が『出たくない』と言った時​

  • 「出ないなら早く言って！ 止めるつもりなら、止める側の責任もあるでしょ」

台詞運用メモ​

  • 相手の名前を呼ぶ回数が少ない方がらしい
  • 千紘を嫌われキャラにしない最重要ルール: 言葉は厳しいが、行動は常に相手のためにしている。この逆転をプレイヤーに早期に見せること
  • 皮肉やドライユーモアは「怒り」の場面では出さない。余裕がある日常場面でだけ使う
  • 不器用な気遣いの描写を千紘の厳しい言葉より先にプレイヤーに見せることで、後の厳しさに対する印象が変わる
  • 鎧が外れる瞬間は章あたり1回に絞る。頻繁だとテンプレになる
  • 千紘の可愛さは「本人が自覚していない」ことで成立する。自覚した瞬間にキャラが崩れる

名前の採用理由​

  • 四辻 は硬質で管理側の印象を持たせやすい
  • 千紘 は漢字にすることで先輩としての芯を残しつつ、読みは柔らかく保てる
  • 姓呼びでも名呼びでも、先輩らしい重さがある
