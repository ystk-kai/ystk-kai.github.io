キャラクターデザイン・原画の方針

「よるのうたかた」のキャラクターデザイン・原画は、FrontWing の フミオ と さいね の画風・彩色を参考とする。

フミオ​

うたシリーズ全作（ゆきうた・そらうた・ほしうた）の原画担当。

画風の特徴​

  • 癖の少ない端正なキャラクターデザイン: 極端なデフォルメや強い個性よりも、幅広い層に受け入れられやすい万人向けの整った造形
  • 一貫した「フミオテイスト」: 作品を跨いでもキャラクター描写に統一された作家性がある
  • 作品の世界観に寄り添う柔軟性: 冬の儚さ（ゆきうた）、瀬戸内の静けさ（そらうた）、夏の学園もの（ほしうた）まで幅広く対応

彩色の特徴​

  • ギャルゲ塗りの正統派: アニメ塗りをベースに、エアブラシやグラデーションで柔らかさを加えた仕上がり
  • 高彩度でカラフルな色使い: 下塗りと影色の色相差を大きくとり、華やかな印象を与える
  • グラデーションを活用した柔らかい陰影: パキッとした塗りとなめらかな塗りの調和
  • 透明感のある表現: レイヤー効果（加算・発光など）による透き通るような肌や髪の表現
  • 色トレスによる線画のなじませ: 線画を周囲の色に溶け込ませ、絵全体に統一感と柔らかさを持たせる

主な参加作品​

  • うたシリーズ（ゆきうた / そらうた / ほしうた / ほしうた SS）— 原画・キャラクターデザイン
  • 智代アフター ～It's a Wonderful Life～（Key）— 原画
  • ほしうた コミカライズ（月刊コンプエース）— 漫画

さいね​

ボカロ・VTuber 系イラストレーターとして台頭し、2025年に FrontWing 初参加（『花束を君に贈ろう -Kinsenka-』キャラクターデザイン/原画）。

画風の特徴​

  • 華やかで温かみのあるイラスト: カラフルなファッションデザインと柔らかく暖かい光の描写が最大の特徴
  • 柔らかく瑞々しい描写: ゲーム原画としてもその質感が活かされている

彩色の特徴​

  • 影色や環境光の反射に紫を多用: 配色だけで「さいねの絵」と識別できるほどの独自性
  • 鮮やかで明るいトライアド配色: アクセントカラーの使い方が印象的
  • 紫のオーバーレイと逆光表現: 幻想的な光の演出
  • 色彩の鮮やかさへの強いこだわり: 画集では「カレイド印刷」を採用するほど

主な参加作品​

  • 花束を君に贈ろう -Kinsenka-（FrontWing、2025年）— キャラクターデザイン/原画
  • 鈴原るる（にじさんじ）— キャラクターデザイン
  • 終焉ノ栞プロジェクト — キャラクターデザイン・小説挿絵
  • After the Rain（そらる × まふまふ）CD ジャケット
  • 画集『world×world』（KADOKAWA）、『COLORS ROOM』（玄光社）

「よるのうたかた」への適用方針​

フミオの 癖のない端正な造形 と ギャルゲ塗りの正統派彩色 をキャラクターデザインの基盤としつつ、さいねの 紫を基調とした幻想的な光の演出 と 鮮やかな配色 を夜の世界観の表現に取り入れる。
