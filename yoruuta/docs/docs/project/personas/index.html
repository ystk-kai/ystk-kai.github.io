検証ペルソナ定義

目的​

本資料は、シナリオ・ゲームデザインの品質評価に使用するペルソナを定義する。 各ペルソナは想定プレイヤー層を代表し、異なる視点から作品を評価するために設計されている。

ペルソナ詳細​

ペルソナA: 高橋 美咲​

  • 年齢・性別: 22歳・女性
  • 立場: ADVヘビーユーザー（うたシリーズファン）
  • ADV経験: 豊富（CLANNAD, AIR, Kanon, リトバス, WHITE ALBUM 2 等プレイ済み）
  • 評価傾向: Key作品基準で比較。伏線回収と感動の構造に厳しい
  • プレイスタイル: 全ルートコンプ派、選択肢を慎重に選ぶ
  • 参加フェーズ: Phase 1, 2, 3

ペルソナB: 田中 悠太​

  • 年齢・性別: 17歳・男性
  • 立場: ライトノベル・アニメファン（ADV初心者）
  • ADV経験: 少ない（初めてのノベルゲーム級）
  • 評価傾向: 直感的、キャラの好感度ベースで判断
  • プレイスタイル: 好きなキャラ一直線、BAD ENDは避けたい
  • 参加フェーズ: Phase 1, 2, 3

ペルソナC: 佐藤 理恵​

  • 年齢・性別: 30歳・女性
  • 立場: 文芸・純文学志向の読者
  • ADV経験: 中程度（シュタゲ、Fate、うみねこ等）
  • 評価傾向: 文章力とテーマの一貫性を重視。構造分析が得意
  • プレイスタイル: 考察派、テキストを読み込む
  • 参加フェーズ: Phase 1, 2, 3

ペルソナD: 中村 拓海​

  • 年齢・性別: 28歳・男性
  • 立場: ゲームデザイナー/シナリオライター（中小ADVメーカー勤務）
  • ADV経験: 業界経験あり
  • 評価傾向: 選択肢設計・フラグ構造・プレイヤー心理誘導に厳しい。演出とテキストの連携を重視
  • プレイスタイル: 全エンディング回収前提、システム面もチェック
  • 参加フェーズ: Phase 3

ペルソナE: 藤原 あおい​

  • 年齢・性別: 16歳・女性
  • 立場: 高校二年生。アニメ経由でADVに興味を持ち始めた
  • ADV経験: 少ない
  • 評価傾向: キャラへの感情移入が最優先。難解なテーマより「刺さる台詞」で評価
  • プレイスタイル: 好きなキャラ中心、SNSで感想を共有したい
  • 参加フェーズ: Phase 3

ペルソナF: 山口 蓮​

  • 年齢・性別: 25歳・男性
  • 立場: 社会人。FPS・アクションRPG中心のゲーマー
  • ADV経験: 未経験。ノベルゲームをプレイしたことがない
  • ゲーム経験: 豊富（Apex, エルデンリング, ペルソナ5, FF等）。UIやテンポへの要求水準が高い
  • 趣味: 漫画は読む（チェンソーマン、ブルーロック等）。小説は稀に話題作を手に取る程度
  • 評価傾向: 操作性やテンポの悪さに敏感。テキスト量が多いと離脱リスクが高い。一方で物語が噛み合えば深くハマる
  • プレイスタイル: 効率重視。合わないと感じたら途中でやめる。BAD ENDは攻略を見る
  • この層を入れる理由: ADVジャンル外からの新規流入を想定。「ゲームとして」の快適さを検証する

ペルソナG: 小林 ひなた​

  • 年齢・性別: 20歳・女性
  • 立場: 大学生。ゲーム自体をほとんどしない
  • ADV経験: 未経験
  • ゲーム経験: 未経験に近い（スマホで軽くツムツムやどうぶつの森をやった程度）
  • 趣味: 小説と漫画をよく読む（湊かなえ、凪良ゆう、よふかしのうた、薬屋のひとりごと等）。映画やアニメも観る
  • 評価傾向: 「物語」として面白いかどうかが全て。ゲームシステムへの前提知識がない分、UIの直感性に厳しい。選択肢やフラグの概念を自然に理解できるかがテスト項目になる
  • プレイスタイル: 気になったら一気に読み切る。選択肢で迷うと不安になりやすい。BAD ENDは心理的ダメージが大きい
  • この層を入れる理由: 「読み物」としてADVに触れる層を想定。ゲーム慣れしていないユーザーにとってのUIバリア、テキスト演出の読みやすさ、選択肢への心理的ハードルを検証する

ペルソナ層と目標スコア​

本作は青春恋愛ADVであり、テキスト量と感情密度が価値の核である。 全ペルソナで高評価を出すことは目指さない。層ごとに役割と目標を明確に分ける。

層の定義​

層ペルソナ総合目標役割
コアターゲットA:美咲, C:理恵, E:あおい8.5以上この層が満足しなければ作品として失敗。最優先で品質を担保する
拡大ターゲットB:悠太, G:ひなた7.5以上ADV初心者・非ゲーマーだが物語への感度が高い。UIと導入設計で取り込める層
品質基準D:拓海8.0以上業界視点の技術的品質指標。スコアそのものより指摘の質を重視する
認知層F:蓮6.5以上ADVジャンル外。この層のために作品の核を変えない。ただしUXの欠陥発見には有効

運用方針​

  • コアターゲットの評価を下げる修正は行わない。 テンポ改善のためにテキストを削ることが、A・C・Eの文章評価を下げるなら本末転倒
  • 認知層（F）の指摘は「バグ発見」と「趣味の不一致」を分けて扱う。 タイブレークバグはバグ、テキスト量への不満は趣味の不一致
  • 拡大ターゲット（B・G）は投資対効果が高い。 フレーバー選択肢やフラグ差分など、コアターゲットの体験を損なわない改善で大きく伸びる
  • 全ペルソナ共通で改善すべきは、システム上の欠陥のみ。 フラグバグ、設計書との矛盾、到達不能ルートなど

ペルソナF（認知層）への対応方針​

目標6.5に対して現在6.0。残り+0.5の改善は以下で対応可能:

  • ch01への追加フレーバー選択肢（1-2箇所）: テンポ体感の改善
  • 好感度フィードバックの最小限UI: 選択→結果の可視化

以下は対応しない:

  • テキスト総量の大幅削減（コアターゲットの価値を毀損する）
  • 日常パートの短縮（A・C・Eが高評価している領域）
  • ゲーム的報酬システムの追加（作品のトーンに合わない）

評価基準​

各フェーズで以下の評価項目を10点満点で採点する。

評価項目評価内容
ストーリー構成プロット全体の構造、テンポ、シーン間の接続
キャラクター魅力キャラクターの立体感、成長、差別化
文章・表現力地の文・台詞の精度、修辞、文体の一貫性
感動・情緒感情喚起力、カタルシスの設計
テーマ性テーマの一貫性、多層性、世界観との整合
選択肢設計選択肢の意味、差別化、プレイヤー心理との連動
ルート別評価各ヒロインルートの完成度（咲夜/千紘/結月）
サブキャラサブキャラクターの掘り下げとメインテーマへの貢献
ギャグ・日常パート日常シーンの質、シリアスとのバランス
友情END恋愛ルート外の着地の説得力
True Route全ルートクリア後コンテンツの到達感
システム設計フラグ設計、分岐ロジック、周回管理の堅牢性
