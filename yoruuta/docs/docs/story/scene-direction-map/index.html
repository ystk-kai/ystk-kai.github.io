演出対応表

目的​

本資料は、シナリオ本文で使っている背景、BGM、SE の識別子と、その場面意図を対応づける。 演出設計、素材制作、命名統一の基準として使う。

ルート別の演出軸​

咲夜ルート​

  • 主背景: old_music_room_interior, small_hall_evening
  • 主BGM: empty_school, dusk_theme
  • 狙い: 声が閉じた場所から舞台へ移る流れを作る

Aルート終盤演出メモ​

  • 本番導入では、台詞の前に短い無音に近い待ちを入れる
  • 最初の一文までは立ち絵変化を絞り、声が立ち上がったあとに感情を動かす
  • エンディング朝シーンでは、夜の秘密が昼の空間へ持ち込まれた感覚を優先する

千紘ルート​

  • 主背景: faculty_room_corridor, classroom_after_school, small_hall_evening
  • 主BGM: tense_theme, school_after, school_day
  • 狙い: 張り詰めた実務空間から、役割分担の回復へ移す

Bルート終盤演出メモ​

  • 緊張が解ける瞬間は泣き崩れではなく、受け渡しの手元で見せる
  • 台本や飲み物の受け取りを、支えを受け取る変化の象徴として扱う
  • 文化祭翌日の昼は、音楽よりも平常の教室音を意識して平熱へ戻す

結月ルート​

  • 主背景: town_street_night, library_evening, town_street_evening
  • 主BGM: night_walk, dusk_theme, empty_school
  • 狙い: 停止した時間から、閉じない選択へ少しだけ動かす

Cルート終盤演出メモ​

  • 本番の四行は長く語らせず、短さそのものを勇気として見せる
  • 終幕では夜景を強調しすぎず、夕方から夜へ移る途中の色を残す
  • 前向きさの強調より、保留をやめた静かな変化を優先する

実装上の注意​

  • シナリオで使う識別子は本資料を基準に増やす
  • 背景やBGMを追加する場合は、先にここへ意味を追記する
  • 背景とBGMは感情導線そのものなので、同義語の乱立を避ける
