章イベント詳細

目的​

本資料は、共通ルートと個別ルートにおける章ごとのイベント目的、感情の進行、主要シーンを整理する。 執筆時のアウトライン、および実装時のシナリオ分割単位の基礎資料として使う。

シーン単位の詳細は ストーリートリートメント と シーン分解表 を基準とし、本資料は章ごとの圧縮版として扱う。

物語内の時系列順​

章番号は設計上の分類であり、プレイ順とは異なる。物語内の時系列は以下の通り。

プレイ順章時期感情の位置づけ
1第0章 夜の入口Day 1出会い・空気の定義
2第1章 昼の顔、夜の顔Day 2〜5参加者の出揃い
3第2章 企画の立ち上げDay 5〜10企画始動・場の形成
4第5章 放課後の輪郭Day 10〜20日常の蓄積（幸福の基盤）
5第3章 夜の露出Day 15〜22傷の表面化
6第6章 交差する視線Day 20〜28ヒロイン同士の関係
7第7章 嵐の前の凪Day 28〜32幸福の頂点
8第4章 破綻寸前Day 32〜35共通ルート最大の圧
9個別ルートDay 35〜ルート分岐後

※ 第5章と第3章、第6章は時期が重なる部分がある。日常の蓄積と傷の表面化が並行して進む構造。

共通ルート​

第0章 夜の入口​

章の目的​

  • 作品の空気を定義する
  • 主人公とヒロインAを出会わせる
  • 「夜」が感情の名前であることを暗示する

主要シーン​

  • 放課後、主人公が旧音楽室で歌声を聞く
  • Aが人に聞かれたくなかったと反応する
  • 主人公はその場を無理に掘らず、短い会話だけで終える
  • 帰り道で主人公の独白を入れ、「聞いてしまった声」が残る

仕込むもの​

  • 主人公が音や声に敏感であること
  • Aが「届くこと」を怖がっていること
  • 旧音楽室が秘密の場所として機能すること

第1章 昼の顔、夜の顔​

章の目的​

  • 3ヒロインを昼の学校側で見せる
  • 主人公が「昨日見た顔」と「今日の顔」の差に気づく
  • 物語の参加者を出揃わせる

主要シーン​

  • Aと学校で再会するが、まるで何事もなかったように振る舞われる
  • Bが文化祭実行の議論で強い言葉を使い、主人公と軽く衝突する
  • Cが放課後の教室や通学路で、熱のない穏やかさを見せる
  • 主人公が3人とも「夜に別の顔を持つ」と感じ始める

仕込むもの​

  • Bの怒りは表面だけでないこと
  • Cの静けさが楽観ではなく諦めに近いこと
  • 主人公が無理に距離を詰めないこと

ミニゲーム導線​

  • 湊サブ到達時、放課後の寄り道として arcade_stack を解放する
  • 本編導線ではなく、最初の宝探し的発見として扱う

第5章 放課後の輪郭​

章の目的​

  • 日常の積み重ねにより「この時間がかけがえのない」感覚を育てる
  • 3人のヒロインとの距離が自然に縮まっていく過程を描く
  • 企画が始まる前の「普通の放課後」の価値を読者に刻む
  • 各ヒロインの「温度が上がるモード」を日常の中で自然に見せる

主要シーン​

  • 中庭昼食の定例化（朔・湊・結月・あかり）
    • 結月の意外な知識が出る（食べ物の産地、本の感想など）。あかりが「何で知ってるの」と反応
    • 結月の「ハンバーグにはポン酢」のような小さなこだわりが出る場面
  • 旧音楽室での綾瀬川との穏やかな時間
    • 歌わず、ただいる。だが咲夜が無意識に鼻歌を口ずさんでいることに朔が気づく
    • 指摘された咲夜の照れ反応（「……え。歌ってた？ ……ごめん、癖で」）
    • 企画の言葉選びについて咲夜が珍しく具体的に口を出す場面（創作こだわりモード初出）
  • 千紘の実行委員会を外側から観察（有能さと余裕のなさ）
    • ただし千紘が朔の作業の間違いを黙って直していた痕跡を発見する場面を入れる
    • 廊下の会話でドライユーモアが出る（「予定通りに遅れてるなら…」）→ 千紘が怖いだけではないと気づく
  • 湊とゲームセンターへ寄り道（ミニゲーム導線）
  • 雨の日の図書室で結月と同じ空間にいる安心感
    • 結月が読んでいる本について不意に語り出す（「途中で一回怒ると思う」）
  • 千紘との廊下の会話（「論点が通じる相手」として認識される）
    • 千紘が自販機で買ったもう一本を「二つ出ただけ」と朔に渡す
  • 夕焼けの通学路で綾瀬川と少しだけ歩く（無言が不快でなくなる）
    • 咲夜が「――あ。雨の匂い、する」と言う。感覚の鋭さが自然に出る
  • 週末の自宅で真帆に「最近楽しそう」と言われ変化を自覚

仕込むもの​

  • 結月の「また明日」が当たり前になる安心感（後の喪失との対比）
  • 綾瀬川が旧音楽室以外の場所でも自然に存在できるようになる変化
  • 千紘が「使える人間」として朔を認識する布石
  • 真帆の「楽しそう」指摘が第7章で反復される伏線
  • ヒロインの温度モードの初出: 咲夜の創作こだわり、千紘の不器用な気遣い+ドライユーモア、結月の知識フラッシュ+小さなこだわり。これらが第4章で消えることで喪失感が増す
  • 殊羽の日常コメディ: 中庭昼食に途中から合流し、結月の地元ネタに食いつく。あかりとの掛け合い（「じゃん」vs「〜と？」）で場の温度が最も上がる。第4章でこの空気が消えることとの落差を作る

第2章 企画の立ち上げ​

章の目的​

  • 文化祭の小規模朗読企画を具体的に立ち上げる
  • 主人公が受け身から実務担当へ移る
  • 3人が同じ場に集まる理由を作る

主要シーン​

  • 自由企画枠の話が出る
  • Aの断片的な言葉、Bの反発、Cの投げやりな賛成が交差する
  • 主人公が音響や進行で手伝うことになる
  • 小講堂を使うための準備、書類、下見が始まる
  • 3人が企画の意味をそれぞれ違う捉え方で語る

仕込むもの​

  • Aは企画を望んでいるが前に出られない
  • Bは企画そのものを危ういと見ている
  • Cは企画に意味があるのか半信半疑
  • 殊羽が投書箱の設置と管理を引き受ける。図書委員としての経験（匿名の言葉を扱った実績）から自然に手を挙げる。設置作業で箱の左右を間違える天然を見せつつ、「投書に何書いたかは、聞かんよ」と素朴に宣言する。この一言が第4章の喪失と Cルートの回収の起点になる

ミニゲーム導線​

  • scene_02b_minato_iroha_banter 到達時、講堂準備の軽口から card_21 を解放する
  • いろはの静かな観察と、湊の軽さの噛み合わせを「短い判断ゲーム」の入口として見せる

第3章 夜の露出​

章の目的​

  • 3人の傷を表面化させる
  • 企画が「準備していれば形になる」ものではないと示す
  • 主人公に最初の選択を積ませる

主要シーン​

  • Aの原稿か歌詞の断片を主人公が聞いてしまう
  • Bが家庭か過去の件で感情を爆発させる
  • Cが進路や将来について無関心を装う
  • 主人公がそれぞれに別の形で関わる夜イベントが発生する
  • ここで好感度ではなく、信頼の積み上げを感じさせる選択を置く

仕込むもの​

  • 言葉にすることが必ずしも救いにならないこと
  • 主人公の優しさが、ときに踏み込み不足でもあること

ミニゲーム導線​

  • scene_05_preparation の準備風景から lucky_reels を解放する
  • あかりの「見せ方」と場の明るさを、ぼんやり回せる休憩台の入口として扱う

第6章 交差する視線​

章の目的​

  • 朔を介さないヒロイン同士の関係を見せる
  • 企画の準備が深まる中で、投書の選別作業が3人それぞれの傷に触れる
  • 第4章で「この場全体が壊れた」喪失感を作るために、場の厚みを増す

主要シーン​

  • 殊羽が投書を回収・整理して選別の場に持ち込む。投書の封を開けるのは千紘たちだが、殊羽は中身を読まずに手渡す。「預かっとるだけやけん」と軽く言うが、この誠実さが企画の信頼の土台になっている
  • 投書の選別作業で千紘が咲夜の言葉選びの鋭さに気づく（「……使える」と一言）
    • 千紘が咲夜を「感情的な人」ではなく「言葉に対して正確な人」として認識する
  • 結月が千紘の疲労に気づき、自販機の飲み物を千紘の机に黙って置く
    • 千紘は気づいているが何も言わない。「優しさの檻」の結月と「管理の鎧」の千紘の、言葉のない交差
  • 投書を読む作業で、咲夜が結月だけ何も書いていないことに気づく
    • 言葉にはしないが、「この子は書けないんだ」と感覚的に分かる
  • 朔は投書を読む中で、自分が投書できないこと（＝自分の言葉がないこと）に無意識に気づく
    • ここでは気づきを明示しない。投書箱の前で少し止まって、通り過ぎるだけ

仕込むもの​

  • ヒロイン同士の関係を朔の外で成立させることで、「朔がいなくてもこの場は存在した」という重み
  • 朔の「自分の言葉がない」問題のTrue Route伏線（無自覚レベル）
  • 投書の選別を通じて、企画の内容が3人の傷に近づいていく構造

第7章 嵐の前の凪​

章の目的​

  • 最後の平穏を描く。失われる直前の幸福
  • 企画が「形になっている」実感を全員が共有する
  • 読者に「この時間がなくなるのが惜しい」と感じさせる

主要シーン​

  • リハーサル通し（全員集合、紙コップ休憩が幸福な温度で再登場）
    • 殊羽が紙コップを配る時に自分の分を忘れる。湊がいじり、殊羽「なんで笑うと！」。場が最も温かい
    • 千紘が砂糖を多めに入れているのをあかりに見つかり、「効率の話をしているの」と返す。全員が笑う
    • 咲夜がリハーサルの台本の言葉にこだわりを出す（「ここ、声に出すと長い。削っていい？」）。千紘が「……そこは任せる」と初めて咲夜の判断を認める
    • 結月が全員の紙コップの砂糖の量を正確に覚えている（「湊は三つ、あかりちゃんは一つ、久瀬くんは二つ」）。止まっていても周りを見ていた証拠
  • 綾瀬川の笑顔（「少しだけ、楽しかった」と小声）
  • 千紘の軟化（語調がいつもより少し柔らかい、棘のない「うるさい」。朔が遅くまで残ったことに「帰りなさい」と言いつつ引き継ぎメモを置く）
  • 結月の「明日」（自分から「明日のリハーサルも来る」と言う）
  • 全員での帰り道（分かれ道で一人ずつ減っていく）
  • 湊の約束（文化祭後の打ち上げ提案、未来の約束）
  • 前夜の静けさ（真帆が「楽しそうだね」2度目の指摘）

仕込むもの​

  • 紙コップ休憩の再登場で日常の信頼が完成したことを示す
  • 綾瀬川の「楽しい」が第4章で奪われる伏線
  • 千紘の軟化が崩壊後の硬化と対比される
  • 結月の「明日」が予定への消極性からの変化（後の停滞との落差）
  • 湊の打ち上げ約束が果たされるかどうかの緊張感
  • 真帆の2度目の「楽しそう」が朔の変化の最終確認

第4章 破綻寸前​

章の目的​

  • 共通ルート最大の圧を作る
  • 企画の継続と各人の心の安全が両立しない局面を作る
  • ルート分岐の意味を成立させる

主要シーン​

  • 企画内容の一部が意図しない形で外に漏れる
  • 周囲が「重い」「面倒」「自意識」と面白半分に消費する
  • Aは出演をやめようとする
  • Bは企画そのものを潰そうとする
  • Cは静かに身を引こうとする
  • 主人公は、企画を守るか、誰か一人の夜を優先するかで揺れる
  • 最後の夜に誰に会いに行くかでルート分岐する

分岐前の到達条件​

  • そのヒロインとの信頼イベントを一定数満たしている
  • そのヒロインの核心を一度は見ている
  • 主人公がその人を「気になる相手」ではなく「守りたい相手」と認識している

個別ルート​

Aルート前半​

章の目的​

  • Aの罪悪感の由来を明かす
  • 声を他人へ渡すことへの恐怖を現実的に理解させる

主要シーン​

  • 旧音楽室での再対話
  • 過去の失敗について語られる
  • 主人公が「歌えばいい」とは言わないことが重要になる

Aルート後半​

章の目的​

  • Aが自分の言葉を自分の責任で届ける決断をする

主要シーン​

  • 本番直前の撤退寸前
  • 主人公は止めるのでなく、選ばせる
  • 終幕でA自身の声が届く

Bルート前半​

章の目的​

  • Bの怒りが防衛反応であると明かす
  • 主人公が安易な慰めを失敗する

主要シーン​

  • 家庭事情または生活の負荷が見える
  • Bが助けを拒絶し、主人公が一度距離を見誤る
  • 対立のあとで初めて本音が漏れる

Bルート後半​

章の目的​

  • Bが助けられる側になることを受け入れる

主要シーン​

  • 本番前に一番大きな衝突がある
  • 主人公は説得ではなく、逃げ道のない誠実さを示す
  • Bが怒りではない言葉を選ぶ

Cルート前半​

章の目的​

  • Cの諦めが現実的な喪失から来ていると明かす
  • 主人公が「励ます」ことの無力さを知る

主要シーン​

  • 夜の帰り道や静かな場所で、未来を捨てた理由が語られる
  • 主人公の善意が空回りする
  • それでも隣にいることだけは残る

Cルート後半​

章の目的​

  • Cが希望そのものではなく、未来に触れる意思を取り戻す

主要シーン​

  • 本番参加か辞退かで最後まで揺れる
  • 大きな告白ではなく、小さな決断がクライマックスになる
  • 終幕は静かだが最も長く余韻が残る

サブストーリー・日常イベント候補​

共通ルートの厚みを増し、日常の蓄積（＝後の喪失の痛み）を強化するための追加イベント候補。 全てを実装する必要はない。シナリオの密度とバランスを見て取捨選択する。

千紘の弟の電話第3章千紘（+朔が偶然聞く）千紘が弟に「冷蔵庫の二段目」「宿題はテーブルに出した」と電話している。声のトーンが教室とまるで違う。朔は声をかけずに通り過ぎる千紘の家庭の窓。「一人で片づけた方が早い」の背景を行動で見せる
朔×真帆の朝食第5章朔・真帆朝食のトーストを焼く短い場面。真帆が「最近帰り遅いね」→朔「手伝いがあって」→真帆「楽しそう」。二人の距離感（近いが深くない）を日常の一瞬で見せるTrue Routeの「朔が自分のことを語らない」伏線を家庭の空気で仕込む
結月×殊羽の図書室第5章結月・殊羽図書委員の仕事で二人きりになる。殊羽が本の配架を間違え、結月が黙って直す。殊羽「穂坂先輩って、怒らんと？」結月「……怒るほどのことじゃないから」。殊羽が「先輩、本の話だけちょっと声大きくなるの知っとう？」と言い、結月が少し照れるCルートで殊羽が結月の四行を預かる接続の布石。結月が「空っぽではない」ことを後輩視点で見せる
いろは×咲夜の機材搬入第6章いろは・咲夜照明機材を二人で運ぶ。いろはが咲夜に話しかけたいが、言葉が見つからない。咲夜が「……重い？」いろは「大丈夫です。……あの、先輩、」（言いかけてやめる）。咲夜は気づいているが追わないA隠し（いろは視点で咲夜の歌を聞く）への伏線。年下からの憧れ

サブストーリーの設計原則​

  • サブストーリーは共通ルート内に自然に配置する。独立した脇道ではなく、章の主要シーンの前後に挟む
  • 1つのサブストーリーは**1〜3シーン（5〜15分程度）**で完結する。長くしない
  • 好感度への影響は控えめ。サブストーリーの目的は日常の蓄積と人物理解であり、ルート誘導ではない
  • 笑いと伏線の複合を意識する。笑える場面にさりげなく仕込む（コメディ直後の伏線配置）
  • 全候補を実装するのではなく、各章2〜3個を目安に選択する
  • ヒロインのサブストーリーは朔を介さない関係性も見せる（第6章の設計思想と一致）

実装メモ​

  • 各章の終わりに、小さな選択で信頼を積ませる
  • 共通ルートでは、同じ夜でも相手ごとに温度が違うことを見せる
  • ルート突入後は、共通ルートのイベントを流用しすぎず、意味を変えて再使用する
