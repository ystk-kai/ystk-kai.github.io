ペルソナ評価データ

備考

ペルソナの定義は検証ペルソナ定義を参照。本資料は各フェーズの評価結果を記録する。

1. ペルソナ定義​

ペルソナA: 高橋 美咲（22歳・女性）​

  • ADV経験: 豊富（CLANNAD, AIR, Kanon, リトバス, WHITE ALBUM 2 等プレイ済み）
  • 評価傾向: Key作品基準で比較。伏線回収と感動の構造に厳しい
  • プレイスタイル: 全ルートコンプ派、選択肢を慎重に選ぶ

ペルソナB: 田中 悠太（17歳・男性）​

  • ADV経験: 少ない（初めてのノベルゲーム級）
  • 評価傾向: 直感的、キャラの好感度ベースで判断
  • プレイスタイル: 好きなキャラ一直線、BAD ENDは避けたい

ペルソナC: 佐藤 理恵（30歳・女性）​

  • ADV経験: 中程度（シュタゲ、Fate、うみねこ等）
  • 評価傾向: 文章力とテーマの一貫性を重視。構造分析が得意
  • プレイスタイル: 考察派、テキストを読み込む

3. Phase 1 で特定された主要課題​

  1. 千紘の参加動機が不明確
  2. 真帆の連絡経路が不自然
  3. ルート分岐の閾値が厳しすぎ
  4. 結月の母親の状況が曖昧
  5. 序盤のテンポが遅い
  6. 選択肢が3択に偏り単調
  7. 好感度システムがゲートチェックになっている
  8. 伏線の仕込みが不足
  9. ルート別BAD ENDが未実装
  10. サブキャラの掘り下げ不足

4. Phase 2-3 改善内容まとめ​

Priority 1: ドキュメント整合性修正（9項目）​

設定矛盾やキャラクター情報の不整合を修正。

Priority 2: 日常・交流シーン追加（6シーン）​

序盤テンポ改善とキャラクター掘り下げのための新規シーンを追加。

Priority 3: 伏線挿入（8箇所）​

ボイスメッセージやダブルミーニングを活用した伏線を共通ルートに配置。

Priority 4: 選択肢リデザイン（15ファイル）​

  • 沈黙選択肢の導入
  • 対話型選択肢への変換
  • 優しさの罠（一見良さそうな選択が裏目に出るパターン）
  • 3択偏重の解消

ルート分岐リデザイン​

  • 好感度ベースの自動判定方式に変更（「誰についていくか」の明示的選択を廃止）

Priority 5: 湊サブストーリー（3シーン）​

サブキャラ・湊の掘り下げシナリオを追加。

Priority 6: フレンドシップエンディング（3シーン）​

特定ルートに入らなかった場合の友情エンディングを実装。

Priority 7: いろは/水城サブストーリー（4シーン）​

サブキャラクターの個別ストーリーを追加。

Priority 8: True Route（7ファイル）​

全ルートクリア後に解放されるトゥルールートを実装。

Priority 9: 早期BAD END（1シーン）​

共通ルート中の脱落エンディングを追加。

ルート別BAD END（3シーン）​

各ヒロインルートにBAD ENDを実装。

隠しEND（3シーン）​

高好感度報酬としての隠しエンディングを実装。

共通して高評価だった点​

  • トゥルールートの音声メッセージ: チョコミント+連立方程式の組み合わせで泣かせる構造が全員に刺さった
  • 選択肢の「善意トラップ」設計: 一見正しそうな選択が裏目に出るパターンが、Phase 1 の単調さを完全に払拭
  • 友情ENDの「選ばない」ことの肯定: 恋愛ルートに入らないことを否定せず、友情として着地させた構成
  • 結月ルートの「正しさでは届かない」テーマ: 正論では救えない関係性の描写が3名とも高く評価
  • サブキャラ（湊・御子柴・水城）の立体化: Priority 5/7 の追加シーンにより、脇役が物語を支える存在に昇華

共通して指摘された残課題​

  • 千紘ルートの相対的な薄さ: 改善は認められるものの、咲夜・結月と比較すると依然として印象が弱い
  • 御子柴の活用余地: サブストーリーで掘り下げられたが、本編への還元がまだ不十分
  • 真帆の背景掘り下げ: 連絡経路の不自然さは解消されたが、キャラクターとしての深みに余地あり
  • 演出パターンのバリエーション: テキスト面は大幅に改善したが、演出指示の多様性にはまだ改善の余地

ベストシーン（3名合計）​

  1. True Route scene_05 voice_message — 全員が1位に選出。チョコミントと連立方程式の伏線回収が涙腺を直撃
  2. 友情END scene_03 night_road — 2名がTOP3に選出。「選ばなかった」ことを祝福する夜道の会話
  3. 結月ルート scene_02 present_stall「正しさでは届かない」 — 2名がTOP3に選出。善意の限界を突きつける転換点
  4. 結月Hidden END「出してきた」「うん」「うん」 — 美咲が2位に選出。最小限の言葉で最大の感情を伝える構成

Phase 3 再評価結果​

Phase 2 で残っていた課題（中弛み、分岐の感情的接続、サブキャラ活用、選択肢の差別化）に対する改善、および新規コンテンツ（湊×御子柴ギャグパート、いろは/水城サブストーリー、隠しEND）を反映した状態での再評価。

ペルソナ拡張​

Phase 3 では既存3名に加え、2名のペルソナを追加して評価の多角性を確保した。

ペルソナD: 中村 拓海（28歳・男性）​

  • 立場: ゲームデザイナー/シナリオライター（中小ADVメーカー勤務）
  • 評価傾向: 選択肢設計・フラグ構造・プレイヤー心理誘導に厳しい。演出とテキストの連携を重視
  • プレイスタイル: 全エンディング回収前提、システム面もチェック

ペルソナE: 藤原 あおい（16歳・女性）​

  • 立場: 高校二年生。アニメ経由でADVに興味を持ち始めた
  • 評価傾向: キャラへの感情移入が最優先。難解なテーマより「刺さる台詞」で評価
  • プレイスタイル: 好きなキャラ中心、SNSで感想を共有したい

P1: 高橋 美咲（ペルソナA） — ADVヘビーユーザー（うたシリーズファン）​

総合スコア: 9.2 / 10

評価コメント: ch01冒頭の「旧音楽室の反芻」（scene_01_classroom L10-11）により、プロローグとの緊張線が途切れなくなった。うたシリーズファンとして最も評価するのは、ch04 route_split における三人の去り際の回想ナレーション（L13-31）で、「千紘の足音は怒りではなかった」「綾瀬川は振り返らなかった」「結月だけが立ち止まった」の三文が、それぞれのルートテーマを先行提示する構造になっている点。WHITE ALBUM 2 の分岐前夜に匹敵する密度がある。隠しEND三本はいずれも高品質で、特に咲夜hidden（中庭の新曲）は「名前を呼ばれて、初めて自分がここにいると知った」の一節が咲夜の名前リビールと美しく呼応する。True Route scene_04 の選択肢テキスト変更（? 指が、再生ボタンの上にある。）は、プレイヤーの身体性を巻き込む設計として秀逸。

Phase 2 からの変化点:

  • 中弛み問題の解消: ch01-02が「何も起きない日常」から「不穏さを孕んだ日常」に格上げされた
  • route_split の感情的厚み: 以前は好感度判定のみだった分岐シーンに、ナラティブとしての重みが加わった
  • 隠しENDの追加: 周回プレイヤーへの報酬として最適な設計。特に千紘hidden の「助けて」は全ルート通じて最も難易度の高い台詞
  • 御子柴の「元舞台側」伏線が ch02 preparation で自然に接続

残る課題:

  • 千紘ルートの本編は依然として咲夜・結月に比べてやや薄い（hidden が強力な分、本編との落差が目立つ）
  • 真帆の掘り下げがTrue Routeに集中しており、共通ルートでの存在感がまだ弱い

P2: 田中 悠太（ペルソナB） — ライトノベル・アニメファン（ADV初心者）​

総合スコア: 9.3 / 10

評価コメント: 湊×御子柴のバンターシーン（scene_02b）が最大のヒット。八十円コーンスープネタの継続、紙コップ裏表の決着、「笑わせるのが上手ですね」からの「笑ってくれる方が楽だからな」の切り返しまで、ギャグとシリアスの温度管理が完璧。初心者にとって最も辛い「全文が重い」問題が、これらのギャグパートで大幅に緩和された。BAD END の罠設計も理解しやすく、特に結月ルートの「一緒に読むよ」が BAD になる理由が「善意の侵入」として直感的に納得できる。ただし選択肢の「正解が分かりにくい」という初心者特有のストレスは依然として存在する。

Phase 2 からの変化点:

  • ギャグシーンの質と量が大幅向上（scene_02b 単体でもコンテンツとして成立する密度）
  • BAD END の罠パターンが「優しさの罠」から「善意の侵入」に差別化され、やり直し時の学びが明確になった
  • ch02 preparation の二択追加で、プレイヤーの心理的な関与が早い段階から発生
  • コーンスープがシリーズ通してのモチーフとして機能し始めている

残る課題:

  • 好感度の可視化手段がないため、初心者はどのルートに向かっているか分かりにくい
  • 隠しENDの条件が複雑で、攻略なしでは辿り着きにくい（ただしこれは設計意図通りとも言える）

P3: 佐藤 理恵（ペルソナC） — 文芸・純文学志向の読者​

総合スコア: 9.5 / 10

評価コメント: Phase 3 で最も評価すべきは、テキストの「地の文の精度」が全体的に引き上がった点。ch15 friendship scene_01 の冒頭「三日が経った」「四日目の昼休み、湊が言った。『で、どうするの』」の時間経過の処理は、短編小説的な切れ味がある。いろは sub_06「光の声」テーマは本作のテーマである「声」を視覚に拡張する試みとして成功しており、「好きだったから、壊れたんです」の一文は登場人物の内面を最短距離で開く文章力を示している。水城先生 sub_08 の「聞く側」定義は本作の主人公像を言語化するものとして極めて重要。iroha_mizuki 交差シーンの「裏方のまま当事者」という命題は、「よるのうたかた」のメタテーマとも読める。結月hidden の「出してきた」「うん」「うん」は、文芸としての到達点。三行で一つの物語を閉じている。

Phase 2 からの変化点:

  • サブキャラクターの物語がテーマの多層性を生んでいる（「声」→「光の声」→「裏方の当事者性」の連鎖）
  • 水城先生が「教師キャラ」の紋切り型を完全に脱却し、独自の存在感を獲得
  • 選択肢テキストの詩的精度が向上（「指が、再生ボタンの上にある。」）
  • 結月・千紘の隠しENDにおける最小限のテキストで最大の感情を伝える技術が顕著

残る課題:

  • 一部のナレーション（特に共通ルート中盤）にまだ説明的な箇所が残る
  • 真帆のキャラクターが「便利な姉」として機能する場面がTrue Route以外で散見される

P4: 中村 拓海（ペルソナD） — ゲームデザイナー/シナリオライター（業界人）​

総合スコア: 9.0 / 10

評価コメント: フラグ設計の質がPhase 3で商業水準に達した。ch03 千紘の三択にchihiro_noticed_fatigueフラグを追加し、「顔色が」選択肢が単なる好感度+1ではなく固有の物語分岐を生むようにした判断は正しい。ch02 preparationのpreparation_pessimistフラグは、主人公の内面状態をゲームシステムに落とし込む好例。route_split の好感度判定ロジック（三者比較の優先順位チェーン）は実装として堅牢で、テスト済みである点も評価できる。GlobalClearDataシステムと14エンディングのフラグマーカーにより、周回管理が設計として完結している。隠しENDの条件設計（高好感度+信頼ステップ+コアフラグの複合条件）は、プレイヤーが「正しいプレイ」ではなく「深いプレイ」を求められる構造として優れている。

Phase 2 からの変化点:

  • 選択肢ごとの固有フラグ追加で「全選択肢に意味がある」設計が実現
  • BAD ENDの罠差別化により、プレイヤーの失敗体験に多様性が生まれた
  • endingDetection分離によりテスト可能性が大幅向上（177テスト全パス）
  • 御子柴の伏線（「裏方だけでは知り得ない知識」の指摘）が自然に配置された

残る課題:

  • 好感度とフラグの相互作用が複雑化しており、デバッグコストが高い（テストカバレッジで補っているが）
  • 演出指示（@chara, @bgm, @wait等）のパターンにもう少しバリエーションがあると、シーン間の差別化が強まる
  • 一部のルート分岐ロジック（three_way_compare等）がフォールスルー的で、意図しない分岐が起きるリスクがゼロではない

P5: 藤原 あおい（ペルソナE） — 高校生（ターゲット年齢層）​

総合スコア: 9.1 / 10

評価コメント: 同世代のキャラクターとして最も共感したのは結月。「無事に過ごせた」という台詞（scene_04_yuzuki_evening）が刺さる。進路の話題で笑顔が固まる描写は、今まさに同じ空気を吸っている身として他人事ではない。湊の「映えより生存」「八十円の方が心の余裕」は友達に送りたい台詞。御子柴の「好きだったから、壊れたんです」は泣いた。隠しENDの結月hidden「出してきた」「うん」「うん」はスクショしてSNSに上げたい。千紘hidden の「助けて」はこのゲームで一番重い台詞だと思う。咲夜hidden の中庭での歌は、最初から聞いてきたからこその感動がある。全体的に「重いけど暗くない」バランスが良く、友達にも薦められる。

Phase 2 からの変化点:

  • ギャグパートの追加で「辛い」だけではなくなった。特にコーンスープネタは定番化してほしい
  • 結月の進路描写が具体的になり、「自分のこと」として読める
  • BAD ENDが「理不尽」ではなく「あのとき私もそう言ったかも」と思える設計になった
  • 隠しENDが「ご褒美」として機能しており、全クリアへのモチベーションになる

残る課題:

  • 水城先生のサブストーリーは、高校生視点だとやや「大人の話」感が強い
  • テキスト量が多く、途中でセーブして中断する場面が増えそう（自動セーブ/しおり機能があると嬉しい）

総合コメント​

Phase 3 の改善は、Phase 2 で残った「接続の弱さ」と「選択肢の意味の薄さ」を的確に補修した。特に以下の3点が全体の品質を押し上げている。

  1. 緊張線の維持: ch01冒頭の旧音楽室反芻、ch04の三者回想ナレーション、ch15の「三日が経った」時間処理により、シーン間の感情が途切れなくなった。これは構造改善として最も効果が大きい。

  2. 選択肢の固有性: 全選択肢に固有フラグ・固有リアクションが付与されたことで、「どれを選んでも同じ」問題が解消した。特に結月ルートBADの「一緒に読むよ」→善意の侵入パターンへの変更は、プレイヤーの失敗体験を「理不尽」から「気づき」に変換する設計として模範的。

  3. サブキャラの物語化: 御子柴の「光の声」、水城の「聞く側」定義、iroha_mizuki交差シーンの「裏方のまま当事者」は、いずれも本作のメインテーマである「声を届ける」の変奏として機能している。サブキャラが独自のテーマを持ちながらメインテーマと共鳴する構造は、商業作品でも稀な達成。

残る全体的課題:

  • 千紘ルート本編の相対的な薄さ（hidden ENDの完成度が高いだけに本編との落差がある）
  • 真帆の共通ルートにおける存在感
  • 演出指示のバリエーション（テキスト品質に対して演出指示がまだ追いついていない）
  • 好感度の可視化（初心者向けのヒント機能の検討余地）

Phase 1（7.0）→ Phase 2（8.9）→ Phase 3（9.2）と段階的に品質が向上しており、特に選択肢設計（5.7→8.2→9.0）の改善幅が最も大きい。現時点でフリーの同人ADVとしては極めて高い完成度に達しており、残課題はいずれもポリッシュの領域にある。
