キャラクター設計 概要

目的​

本資料は、キャラクター設計全体に共通する方針、差別化の軸、命名規則、記号・呼称の運用基準をまとめる。 各キャラクターの個別設計は、キャラクターごとのファイルを参照すること。

設計方針​

  • サブキャラクターは主役を食わない
  • 役割は明確に分ける
  • ギャグ専用、説明専用にしない
  • 主人公やヒロインの弱さを別の角度から照らす存在にする

6人の差別化軸​

メインヒロイン（自分の夜を抱えている）​

  • A 咲夜は隠す
  • B 千紘は拒む
  • C 結月は諦める

サブヒロイン（他者の夜に関わることで自分の夜を回避している）​

  • 殊羽は預かる
  • あかりは支える
  • いろはは見守る

メインとサブは対構造を成す。メインは「自分の声を出せない」、サブは「他者の声に関わることで自分の声を後回しにしている」。朔（True Route）は「聞くことで自分の夜を回避している」であり、サブヒロインと同じ構造を共有する。

この差が立っていれば、見た目や名前は後から決めてよい。

立ち絵全体方針​

  • 今どきの恋愛ADVらしく、ヒロインはロングヘア寄りを基調にする
  • ただし全員を同じ長さにせず、重いロング / 硬い短髪 / 柔らかいロング の3分割で並びを作る
  • 差は髪の長さだけでなく、髪質、分け目、結び方、姿勢、配色で出す
  • 並んだ時に、Aは縦、Bは横、Cは抜け感で認識できることを優先する
  • 主人公は情報量を抑え、3ヒロインの色が立つ中立シルエットに留める

4人の並びの設計​

  • 朔は観測者の中立色
  • 咲夜は沈黙の縦線
  • 千紘は衝突の鋭角
  • 結月は停滞の曲線

この並びにより、立ち絵を横に置いた時点で役割差が見えるようにする。

命名ルール​

  • 著名な既存キャラクターとの同姓同名は避ける
  • 他作品と同じ読みになること自体は許容する
  • 響きより会話上の自然さを優先する
  • 夜モチーフを無理に姓名へ入れない
  • ヒロインの名のひらがな表記は、多くても1人程度に絞る
  • 苗字は少し珍しくてもよいが、初見で読みが推測できる範囲に収める
  • 現代のゲームで違和感なく読める柔らかさを優先する

記号・口語表現ガイドライン​

ADVの台詞は小説と異なり「声が聞こえる文章」を目指す。 記号はキャラクターの呼吸・テンポ・感情の温度を伝える手段であり、書き分けの重要な柱である。

記号の役割定義​

記号役割使用上の注意
!感情の勢い・強調・驚き。叫びだけでなく、断言や呼びかけにも使う連続使用（!!）は原則避ける。1文に1つまで
?疑問・確認・問いかけ。語尾上がりのニュアンスも含む修辞疑問にも使ってよい
......ためらい・沈黙・余韻。言葉を飲み込む間三点リーダーは必ず偶数個（......）で使用
----言い直し・割り込み・思考の断絶ダッシュも偶数個（----）で使用
ー音の引き伸ばし。口語の自然なリズムを作るカタカナ語以外でも口語では積極的に使う
っ促音。言葉の詰まり・勢い・歯切れの良さ小さい「っ」は感情のアクセント
〜甘え・からかい・語尾の揺れ多用すると幼くなるので注意

あだ名・愛称の方針​

  • 主要キャラ同士では、序盤からあだ名を使わない
  • あだ名は、関係性の近さよりも、会話の軽さを出すためにサブキャラ側で使う
  • 朔とヒロインの間では、姓呼びから名呼びへの変化を重要イベントとして扱う

呼称設計の基本方針​

  • 読みやすさよりも、関係性が伝わる呼び方を優先する
  • あだ名は乱用しない
  • 呼称変化はイベントとして使う
  • 主要キャラ4人は、会話だけで誰が喋っているかある程度分かるようにする

呼称で避けること​

  • 序盤から名呼びを多用しない
  • 主要キャラ同士で軽すぎるあだ名を使わない
  • 関係が浅い段階で呼称を変えない
  • 同じキャラが相手によってぶれた呼び方をしない

サブヒロインの共通ルートでの機能​

サブヒロインは個別ルートを持つが、共通ルートおよびメインルート内では以下の機能を維持する。

  • 殊羽: 場の温度計、コメディ起爆剤、投書箱の番人
  • あかり: ツッコミ役、結月の過去の証人、外部視点
  • いろは: 年下の客観視点、実務情報の持ち込み

サブキャラクター同士の役割分担​

  • 湊: 会話の流速
  • 水城: 現実性と外圧
  • 真帆: 家の中の温度
  • 凪: 咲夜ルートの過去の具体性
  • 恒一: 千紘ルートの怒りの根
  • 美弥子: 結月ルートの停滞の現実

サブヒロイン運用方針​

  • 共通ルート・メインルート内: 上記の機能を維持する。メインヒロインの感情の核を奪わない
  • サブヒロインルート内: 自身の「夜」と感動の核を持つ。ただしメインルートの1/3〜1/2の密度に留める
  • サブヒロインルートのクライマックスは文化祭本番ではなく、日常の中の小さな決断に置く

サブキャラ運用方針​

  • サブキャラ（湊・水城・真帆・凪・恒一・美弥子）が結論を言わない
  • ルートの感情の核は必ず主役同士で回収する
  • 親友は便利な相談AIにしない
  • 大人は障害物にも救済装置にもしすぎない
  • 呼称ネタは反復しすぎず、距離感の変化が見える場面で使う

実装メモ​

  • 朔はゲーム画面に立ち絵は表示しないが、人物像の設計資料として立ち絵を作成済み
  • 咲夜は瞬きと伏し目が重要
  • 千紘は眉と口角で緊張を作る
  • 結月は視線外しと微笑の差で停滞を作る
