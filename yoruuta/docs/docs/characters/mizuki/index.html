水城 真琴（みずき まこと）

基本情報​

  • 名前: 水城 真琴（みずき まこと）
  • 立場: 国語教師兼、文化祭の自由企画担当

役割​

  • この上演企画の現実性を担保する大人
  • 学校側の懸念を代弁する
  • ただし一律否定ではなく、条件付きで機会を与える

性格​

  • 理知的
  • 余計な感傷に流されない
  • 生徒の言葉を待てる大人

見た目​

  • 30代前半
  • 落ち着いた紺や灰の装い
  • 派手さよりも整った印象

  • 物語後半では、生徒の覚悟を試す役になる
  • 安易に名呼びへ移行しない
