久瀬 朔（くぜ さく）

基本情報​

  • 名前: 久瀬 朔（くぜ さく）
  • 学年: 高校2年生
  • 立場: 放送委員兼、文化祭の音響補助

物語上の役割​

  • 3人の夜を見つける入口
  • 文化祭の小規模朗読企画を動かす実務役（音響・構成）
  • 聞き手であり、最後には選び手になる

企画への創作的関与​

朔は単なるロジスティクス担当ではなく、音響構成の設計者として企画に創作的な賭け金を持つ。

  • 放送委員としての音響経験を活かし、朗読の間の環境音・音楽・沈黙の配置を設計する
  • 「声をどう届けるか」は朔にとって、「聞けなかった後悔」の裏返しの行為
  • 音の構成は朔の数少ない能動的表現であり、ここにだけ朔の美意識が出る
  • 第4章で「企画を守るか、人を守るか」の選択に、朔自身の作品を手放す痛みが乗る
  • 企画の音響設計が朔自身のアイデンティティと結びついているが、本人はそれを「自分の表現」とは認識していない。この無自覚がTrue Routeで露出する

必須条件​

  • 自分にも小さな夜を抱えている
  • 他人の話を無理にまとめない
  • 受け身から能動へ変化できる
  • 優しさだけでなく、選ぶ責任を背負える

避けること​

  • 万能な理解者にしない
  • 自分の痛みを作品の中心にしない
  • ただ受容するだけで物語を進めない
  • 音響設計を「才能のある主人公」の証拠にしない。朔の音響は技術ではなく、聞くことへの執着の延長

抱えている小さな夜​

中学時代、初恋の相手から夜に届いた長い音声メッセージを、翌日でいいと思って最後まで聞かなかった。 その子は翌朝に家庭の事情で転校し、朔は内容を確認しないまま関係を終わらせてしまった。 途中まで聞いた限りでは日常の話だったが、最後に何を言おうとしていたのかは分からないままだ。

以後、

  • 聞ける時に聞かなかった
  • 相手が伝えようとした気持ちを受け止めなかった

という後悔を抱えている。

そのため朔は、

  • 声や言葉を軽く扱えない
  • だが踏み込みすぎて壊すことも怖い

という二重のためらいを持つ。

深層: 聞く側としての空虚​

表層の「聞けなかった後悔」の下に、より構造的な問題がある。 朔は聞く側に立つことで、自分の傷を回避している。

他人の夜を受け止めることに没頭し、自分の夜――姉・真帆との距離、家庭の空洞、「自分には声にするものがない」という空虚――を直視しない。

  • 真帆は「最近楽しそうだね」と観察するが、朔が自分のことを語らないことには触れない
  • 家に帰っても特に話すことがない。家庭の希薄さを朔は問題だとすら思っていない
  • 「聞く側」であることが朔のアイデンティティになっている

個別ルートでの伏線​

各ルートで、ヒロインが一度は「朔は自分のことを言わない」と指摘する。 朔は「俺のことはいいから」で流す。 この反応は個別ルートでは自然に見えるが、True Routeで構造として露出する。

True Routeでの回収​

三人が先に動いたことで「聞く側」の立場を失った朔に、湊が「お前、結局自分のことは何も言わないよな」と核心を突く。 三人が、各ルートで朔から受け取った態度を朔に返す。

  • 咲夜は朔の沈黙を待つ（Aルートで朔が咲夜にしたこと）
  • 千紘は朔の範囲を問う（Bルートの対等の態度）
  • 結月は黙って隣にいる（Cルートの寄り添い方）

この深層は作品の中心には置かない（「避けること」の制約を維持する）。True Route専用の縦軸として機能させ、個別ルートでは伏線レベルに留める。

性格​

  • 穏やか
  • 沈黙を埋めるために喋らない
  • 人の気配や声の揺れに敏い
  • 引き受けた役目は投げない

見た目​

  • 身長176cm前後
  • ダークブラウンの短髪。光の加減で茶色みが分かる程度
  • 目元はやや切れ長で涼しく、瞳は灰がかった青。無表情だと「怖い顔」と言われるが、実際には整った造りをしている
  • 制服の上に濃いチャコールのカーディガンを羽織ることが多い
  • 派手さはないが顔立ちは端正。本人に自覚がなく、声や佇まいで印象に残るタイプ

一人称​

俺（抑えた話し方なので、荒くならないようにする）

口調​

  • 短文基調だが、一文は完結させる。単語だけで終わらせない
  • 相手の言葉を受けて返す。受けるだけでなく、一手加えて返す
  • 決めつけを避ける
  • 優しいが甘やかさない
  • 主人公として全シーンに出るため、反応が薄すぎると会話の壁になる

場面による応答密度​

朔の短文は「常に最短」ではなく、会話相手と場面に応じて密度が変わる。

場面応答の長さ例
シリアス・傾聴短い。沈黙も武器「……待つ」「言いにくいなら、そのままでいい」
日常会話普通。一文で完結「手伝えるところだけ手伝うよ」
掛け合い・コメディやや長い。ツッコミとして着地させる「善意の窃盗という概念はない」「今後上がる前提で設計するな」

テンポの速い相手（湊・あかり）との掛け合いでは、受け止め＋切り返しで会話を回す。ボケの着地点を作るのは朔の役目。

やってはいけないこと​

  • 単語だけの応答を連続させる（「雑だな」「悪い」「何を」が3連続 → 壁打ちになる）
  • 相手の台詞を繰り返すだけの応答（「入る前だけ？」→ 情報が増えていない）
  • ツッコミを入れるべき場面で黙る（コメディのテンポが死ぬ）

記号使用指針​

記号頻度方針
!稀本当に感情が動いた時だけ。抑制が基本姿勢
ー少ほぼ使わない。朔の言葉は短く切る
っやや少自然な口語として適度に
……中沈黙・傾聴を示す。朔らしさの核
？やや少掛け合いでは相手に問い返して会話を回す手段としても使う

例（シリアス）: 「……待つ」「放っておけないだけだ」「それで終わらせたくない」 例（掛け合い）: 「お前はもう少し静かにできないのか」「その概念はないだろ」「それは今日一番聞きたくない言葉だ」

台詞サンプル​

日常​

  • 「それ、今すぐじゃないなら後でもいい」
  • 「手伝えるところだけ手伝うよ」
  • 「七瀬、お前はもう少し静かにできないのか」
  • 「平気に見えるなら得してる。実際はそうでもない」

掛け合い・ツッコミ​

  • 「善意の窃盗という概念はない」
  • 「今後上がる前提で設計するな」
  • 「字の大きさで解決するな」
  • 「それは今日一番聞きたくない言葉だ」
  • 「人の失敗を連載みたいに言うな」
  • 「その評価は今日だけで二回目だぞ」

ツッコミは受けた内容を具体的に引用して返すのが朔の型。「ひどいな」で止めず、何がひどいか一言足す。

相手が沈黙した時​

  • 「……待つ」
  • 「言いにくいなら、そのままでいい」
  • 「無理にまとめなくていい。聞こえてるから」

ぶつかる時​

  • 「正しいのは分かる。でも、それで終わらせたくない」
  • 「助けるって言い方は違うな。放っておけないだけだ」
  • 「……それは、違う!」（※!は本当に感情が溢れた時のみ）

終盤​

  • 「俺が決めるんじゃない。お前が決めることだ」
  • 「それでも隣にいることはできる」

核心台詞候補​

  • 「救えるとは思ってない。でも、聞かないまま終わりたくない」

場面別比較: 相手が『出たくない』と言った時​

  • 「分かった。……でも、出ない理由だけは聞かせて」

台詞運用メモ​

  • 感情が動いた時ほど相手の名を使うと変化が見える
  • 掛け合いでは「受ける→相手の言葉を引用して返す」が基本リズム
  • 単語応答（3〜5文字）は1ターン置きに留め、連続させない

名前の採用理由​

  • 久瀬 は自然な響きで会話に置きやすい
  • 朔 は夜明け前の新月を連想させるが、直接的すぎない
  • 主人公として、静かさと前進の気配を両立できる
