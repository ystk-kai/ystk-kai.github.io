御子柴 いろは（みこしば いろは）

基本情報​

  • 名前: 御子柴 いろは（みこしば いろは）
  • 学年: 高校1年生
  • 立場: 放送委員の後輩、文化祭当日の音響補助候補

役割​

  • 年下視点から、場の張りつめ方を映す
  • 機材、連絡、備品などの実務情報を自然に持ち込む
  • 将来の追加シナリオや昇格候補として、主役を食わない範囲で印象を残す

性格​

  • 明るすぎず、気配りが先に立つ
  • 先輩相手には礼儀正しい
  • 空気を読みすぎて入る間合いを測る癖がある

背景​

  • 中学時代、校内の合同発表会で咲夜の歌を一度だけ聞いたことがある
  • 直接の面識はほぼないが、前に立つ声 への憧れがその時に残った
  • いろはにとってその記憶は、歌が上手かった ではなく 押しつけないのに届いた 記憶として残っている
  • その後、中学の終わりから高校のはじめにかけて、正体を知らないまま夜に聞いていた匿名音源があり、hidden ではその気配が咲夜の声と繋がる余地を残す
    • その匿名音源のアカウント名は SakuYa。いろはは綴りだけを覚えていて、当初は咲夜本人と結びつけていない
  • 自分はその後、人前に立つ側で一度折れたため、今は 届かせる側 に回っている
  • 咲夜の停滞を見ると、尊敬していた像ごと痛むため、照明づくりに感情が入る
  • ただしその感情は 私が救いたい ではなく、雑に当てたくない / 届き方を壊したくない 方向で出る

見た目​

  • 身長150cm前後
  • 黒髪の肩下ロング、低めの位置で軽くまとめることが多い
  • 制服はきちんとしているが、袖口や名札に少し生活感がある
  • 大人びすぎず、幼すぎない実在感を優先する
  • 主役ヒロインのロングと被らないよう、毛量は軽くして後輩らしい機動力を出す

一人称​

私

口調​

  • 基本は敬語
  • 丁寧だが堅くなりすぎない
  • 素の反応が出ると敬語が崩れる

記号使用指針​

記号頻度方針
！少素の反応が出た時だけ。敬語の枠を超える瞬間
ー少丁寧な話し方なので伸ばさない
っやや中「ちょっと」「やっぱり」など自然な範囲
……中間合いを測る癖。入るタイミングを探っている
？やや中丁寧な確認形。「〜でいいですか？」「〜ですよね？」

例: 「あ――久瀬先輩。すみません、勝手に使ってます」「几帳面じゃなくて、怖いだけです」

咲夜との関係​

→ 詳細は 咲夜の個別ファイル を参照

  • 関係の起点: 中学時代の合同発表会（いろはは客席側、咲夜は前に立つ側）
  • いろは側の記憶: 歌が上手い先輩 より 前に立つのに押しつけない声 として残っている
  • hidden では、当時の学校での歌と、後から夜に聞いていた匿名音源の主が同じだったと静かに繋がる余地を使える
    • SakuYa という綴りの S と Y の不自然な大文字も、後から思い返すと咲夜の字面を半端に隠したものだと分かる
  • いろはが知っているのは 一度聞いた声 までで、凪との出来事や咲夜の核心の詳細までは知らない
  • 感情の芯: 自分が前に出られなかったから、せめて正しく届かせたい

運用​

  • 本編では補助役に徹し、感情の核を奪わない
  • ただし機材や当日運営を通じて主要人物の変化を見届ける
  • 昇格させる場合は、年下らしい憧れではなく、実務の近さから関係を育てる
  • 将来的に hidden epilogue 候補へ広げる場合も、咲夜 / 千紘ルート本編の決着後にだけ開く補助線として扱う
  • 咲夜派生では 昔に一度だけ救われた側 として機能させ、過去の因縁役にはしない
  • 咲夜との関係は 非対称 が重要で、いろはは強く覚えているが、咲夜側はほぼ知らない状態から始める

台詞サンプル​

日常​

  • 「久瀬先輩、視聴覚倉庫の鍵、先に借りておきました」
  • 「四辻先輩、備品表ここです。赤入れあるなら今のうちにお願いします」
  • 「空気読んで黙ってただけです。仕事はちゃんとします」

咲夜に対して少しだけ緊張が出る時​

  • 「綾瀬川先輩、照明位置だけ確認したいです。……すみません、少しだけ前、お願いしていいですか」
  • 「先輩の声、強く当てすぎると違う気がするので。柔らかい方で組みます」
  • 「昔に一回だけ聞いたことあるんです。だから、今日は雑にしたくないです」

緊張がある時​

  • 「今、入っていい場面じゃないのは分かります。でも、これだけは先に言わないと困るので」
  • 「機材って、黙って壊れてくれないんですよ」
  • 「私が言うのも変ですけど、みなさん今日ちょっと怖いです」

素の反応が出る時​

  • 「え、嘘……っ、本当ですか！？」
  • 「あっ――すみません、つい声出ちゃって……」

  • いろは は年下らしい柔らかさを持ちながら、幼く寄りすぎない
  • 追加シナリオで前面に出しても保つ名前の強さがある
