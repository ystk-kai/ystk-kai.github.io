綾瀬川 咲夜（あやせがわ さや）

基本情報​

  • 名前: 綾瀬川 咲夜（あやせがわ さや）
  • 学年: 高校2年生
  • 立場: 同級生、匿名名義で活動を止めている歌い手

物語上の役割​

  • 主人公を物語に引き込む最初のヒロイン
  • 「夜を言葉や声にする」という主題を最も強く担う

感情の核​

  • 罪悪感
  • 自己否定
  • 誰かに届くことへの恐れ

家庭環境​

綾瀬川家​

  • 父: 綾瀬川医院の院長。町で「綾瀬川先生」と呼ばれ、地元で顔が利く。穏やかだが家の体面を気にする
  • 母: 現役の舞台女優。拠点は都市部で、公演や撮影のため家を空けることが多い。田舎町では「女優さんの奥さん」として知られている

家庭が咲夜に与えた影響​

  • 咲夜の容姿端麗・高身長は母親からの遺伝。声の素質も母から受け継いでいる
  • 母への憧れ: 咲夜は幼い頃から母の舞台を観て育った。母が舞台の上で声を出す姿は、咲夜にとって最初の「声で人の心を動かせる」という原体験。歌を始めたきっかけも、母のようになりたいという気持ちが根底にある
  • 母は現役で表に立つ人間だからこそ、娘が匿名で歌を投稿していたことにプロとしての心配で反応した。人前に出ることの代償を身をもって知っている
  • 歌を禁止されたわけではない: 母が言ったのは「なぜ黙ってやっていたの」「やるなら相談してからにしなさい」。父も「歌うな」とは言っていない。ただ、匿名でこっそりやっていたことへの失望と、家の名前への心配を伝えただけ
  • しかし咲夜にとっては、相談したら「ちゃんとやりなさい」と言われるのが怖かった。母のように本気で表に出る覚悟はないのに、中途半端に隠れて歌っていた自分が恥ずかしかった。結果として誰にも禁止されていないのに、自分で歌えなくなった
  • 父は「綾瀬川先生の娘が」「あの女優の娘が」という二重の視線を意識する。悪意はないが、家の名前が行動の枠を作る
  • 母は今も表に立ち続けている → 憧れと、そうなれない自分への距離が日常的にある。母の舞台を観るたびに「きれいだ」と思い、同時に自分の声を嫌いになる
  • 母が家にいない時間が多い → 家庭に華やかさと空洞が同居している。母がいない日の広い家で、それでも声を出せない

禁止されていないことの残酷さ​

この構造が咲夜の夜の核心に直結する。

  • 凪の件: 誰にも「歌うな」と言われていない。だが自分の声が友人を傷つけた
  • 家庭: 誰にも「歌うな」と禁じられていない。だが自分で歌えなくなった
  • 檻は外にはない。全て咲夜の内側にある
  • だからこそ、外から鍵を開けても意味がない。咲夜自身が「自分の責任で声を出す」と決めないと解放されない → Aルートの核

物語上の機能​

  • 家の名前: 田舎町で名前が通る家だからこそ、匿名活動が特定された時の恐怖が大きい。第4章の「あの歌の人」の囁きが家の名前と結びつくリスク
  • 母の存在: 母は敵ではない。現役で表に立つプロだからこそ、娘が準備なく世に出ることを止めた。この「善意の抑圧」が最も痛い。同時に、母が今も輝いていることが咲夜にとっては眩しさと距離になる
  • 母の不在: 公演期間中は家を空ける。咲夜は母がいない家で一人の時間が多い。だがその家でも声は出せない
  • 旧音楽室の意味: 家は広い。だが声を出せない。学校の古い部屋だけが唯一の場所
  • 明示しすぎない: 家庭の経済力は直接描写しない。「綾瀬川先生」呼びや家の広さ、母の過去への言及から推測させる
  • 千紘との対比: 千紘の「生活の重さ」に対して、咲夜は「名前の重さ」。経済格差を前面に出さず、抑圧の形の違いとして扱う

抱えている夜​

中学時代、親しかった友人の弱音を下敷きにして歌を作り、送別会で披露してしまった。 歌は褒められたが、友人からは「私の傷を勝手に歌にした」と受け取られ、関係が壊れた。

これにより咲夜は、

  • 自分の声は救うより先に傷つけるかもしれない
  • 誰かの感情を表現に変えることは暴力かもしれない

と考えている。

さらにその少し後、家族に隠していた匿名での歌投稿も見つかり、

  • 母は「なぜ相談しなかったの」と心配した。「歌うな」とは言っていない
  • 父は「綾瀬川先生の娘が」「母さんの名前にも関わる」と心配した。これも禁止ではない
  • だが咲夜は、母のように本気で向き合う覚悟もないのに隠れてやっていた自分が恥ずかしくなった
  • 誰にも禁じられていないのに、家で声を出すこと自体ができなくなった
  • 母が家にいる時は特に声を出しにくい。プロの耳に聴かれることへの恐怖がある

結果として、家は録音や練習の場所として完全に失われている。

匿名アカウント名は SakuYa。

  • S と Y だけを不自然に大文字にしている
  • 本人にとっては本名を完全に隠したつもりだが、実際には 咲夜 の字面を半端に引きずっている
  • 中学生らしい、少し気取っていて少し雑な隠し方として扱う

現在の状態​

  • 歌うこと自体はやめていないが、匿名アカウントの更新は止めている
  • 学校では誰も、その歌い手が自分だと知らない
  • 家では歌わないし、録音機材も出せない
  • 旧音楽室のような、扉を閉めて短い仮録音や声の確認だけができる場所でだけ声を出す
  • 名前の読みを言い直すことに慣れておらず、誤読されても流すことが多い

時系列整理​

  • 幼少期〜:
    • 母の舞台を観て育つ。「母のようになりたい」が歌の原体験
  • 中学前半:
    • 家で小さく録音を始める。母に聴かせる勇気はない
    • SakuYa 名義で歌の投稿を始める
    • まだ趣味の延長で、誰にも知られていない
  • 中学中盤:
    • 送別会で 抱えている夜 を披露する
    • 同時期に匿名活動も少し伸び始める
  • 中学中盤の後:
    • 家族に匿名活動が見つかる。母が最初に気づく
    • 母の反応は怒りではなく「なぜ相談しなかったの」→ 現役のプロとしての心配。禁止はしていない
    • 父は「綾瀬川の名前で何かあったら」「母さんの名前にも関わる」という方向で心配する。これも禁止ではない
    • だが咲夜は、母のように覚悟もなく隠れてやっていた自分を恥じ、自分で家での録音や練習をやめてしまう
    • 送別会の件と重なり、届くこと と 見つかること が両方怖くなる
  • 高校以降:
    • 旧音楽室を、仮録音と声の避難場所として使うようになる
    • ただし投稿は止まり、人前では歌わない

性格​

  • 静か
  • 反応は遅いが、感情は深い
  • 自分を下げる言い方が癖になっている
  • 一度決めると譲らない芯がある
  • 説明するより先に引く癖があり、自分の名前の読みすら後回しにする

無意識の引力（センターヒロインの設計）​

咲夜はパッケージ中央に配置されるセンターヒロインであり、「静かで怯えた子」だけでは吸引力が足りない。 以下の特性が、恐怖の下にある本来の魅力を見せる。

  • 無意識の鼻歌: 一人でいると思った瞬間、短いメロディを口ずさむ癖がある。本人は気づいていない。指摘されると動揺する
  • 言葉の手触りへの鋭さ: 他人の言葉選びに対して驚くほど具体的な指摘ができる。自分の表現には怯えるが、表現の「質」には譲らない
  • 反応の遅さの裏: 反応が遅いのは鈍いからではなく、正確に受け止めようとするから。その「間」が相手に「ちゃんと聞いてもらえた」感覚を与える
  • 感覚の鋭さ: お茶の濃さの変化、雨の匂い、音の質感の違いなど、五感に対する繊細さが日常の端々に出る
  • 静かな頑固さ: 普段はためらいが多いが、一度選んだ言葉を変えることは絶対にしない。この芯の強さが終盤の決断と繋がる

創作にこだわるモード​

咲夜の恐怖は「声が人に届くこと」だが、表現そのものへの情熱は消えていない（旧音楽室で声を出し続けている事実がそれを示す）。 歌詞の言葉選び、音の響き、表現の手触りについて話す時だけ、普段のためらいが薄くなり言葉数が増える。

  • 人前で歌うことへの恐怖とは別レイヤーの「職人気質」
  • 企画の台本や言葉の議論で表出する
  • 自分の作品については言えないが、他人の言葉の手触りについては口が開く
  • 普段の …… の頻度が下がり、断定的な言い方が出る → このギャップ自体が魅力
  • この面が見える瞬間が「ただの怯えた子ではない」と気づかせる転換点になる

見た目​

  • 身長168cm前後
  • まっすぐな青みのある黒髪、胸下から腰手前まで
  • 前髪は薄めで、顔まわりにだけ細い後れ毛を残す
  • 瞳は淡い灰青で、光が入ると少しだけ透明感が出る
  • 制服は標準的だが、リボンをきっちり結ばず少しだけ緩い
  • 直線的なシルエットで、静けさを出す
  • 配色はネイビー、スモークブルー、白銀

デザイン意図​

  • 3ヒロインの中で最も縦に長いシルエットを担当させる
  • 「儚い」より「音を溜め込んでいる」印象を優先する
  • 目立つ装飾は避け、声の印象で記憶させる
  • ロングヘアの定番感は使うが、量感を抑えて重心を下げ、静かな主役感で差別化する

呼称変化のイベント設計（咲夜ルート）​

  • 朔は長く 綾瀬川 呼びを維持する
  • 中盤で 咲夜 の読みが さや だと知っても、すぐには使わない
  • 咲夜が自分の声を朔に預ける局面で、初めて 咲夜 を さや と正しく呼ぶ
  • 咲夜側は、終盤で一度だけ 朔くん を使い、その後は場面によって戻してもよい

ヒロインAの名は、序盤は名字中心で運用する。 読みの「さや」は中盤で朔だけが知り、終盤の名呼びで初めて表に出す。

初登場​

  • 放課後の旧音楽室
  • 朔が扉越しに歌を聞き、立ち止まる
  • 咲夜は気づいたあと、「聞かなかったことにして」と頼む
  • 初対面では名字だけを名乗り、名の読みはしばらく宙づりにする

主人公との関係進展​

  • 見られたくない部分を見た相手
  • 結果を急がない相手
  • 自分の声を預けられる相手
  • 終盤で初めて、自分の名を正しく読んで呼ぶ相手

サブキャラとの補助線​

いろはとの関係​

  • いろはは中学時代に咲夜の歌を一度だけ聞いており、咲夜を 前に立つ声の象徴 として記憶している
  • そのため、咲夜ルートでは 声を出す本人 と 光で届かせる側 の対比が自然に成立する
  • ただし咲夜本人にとっては過去の重要人物ではなく、知らないところで残っていた受け手 に留める
  • いろはは凪の件や咲夜の核心の全体像を知っているわけではなく、一度聞いた声が残っていた という個人的記憶だけを持つ

いろはとの認識差​

  • 咲夜 → いろは:
    • 放送委員の後輩
    • 丁寧で実務が強い子
    • 自分に少し緊張している理由までは、当初は分からない
  • いろは → 咲夜:
    • 中学時代に一度だけ強く残った相手
    • 前に立つのに押しつけない 声の持ち主
    • だからこそ、今の咲夜が声を怖がっている状態を見るのが少し痛い
    • hidden では、昔に聞いた学校での歌と、夜に聞いていた匿名音源の気配が静かに繋がる余地を残す

このズレが重要で、関係の感動は 昔から知っていた二人 ではなく、 片方だけが強く覚えていたものが、後で静かに返ってくる ところに置く。 さらに、本編では 残っていた までに留め、救われていた と明言するのは hidden 側の感情回収として温存する。

咲夜 × いろは 呼称運用​

  • 基本呼称:
    • いろは → 咲夜: 綾瀬川先輩
    • 咲夜 → いろは: 御子柴さん
  • 呼び方の意味:
    • いろは側の 綾瀬川先輩 には、単なる先輩呼び以上の緊張が少し混ざる
    • ただし露骨なファン口調にはしない
    • 咲夜側は当初そこに特別な意味を感じていない
  • 話し方の温度差:
    • いろはは普段より一段だけ丁寧になる
    • 咲夜は変わらず静かだが、いろはの実務をきちんと信頼する
    • この二人の会話は、親密さより 慎重な尊重 を先に立てる
  • hidden へ繋ぐ時のルール:
    • いろはは 昔に聞いたことがあります までは言ってよい
    • ただし ずっと憧れていました と言い切りすぎない
    • 本編では 残っていました 程度に留め、救われていました は hidden 側の決定打として遅らせる
    • 咲夜には 知らないところで届いていた と返す形を優先する

台詞サンプル​

日常​

  • 「……おはよう。昨日のことは、忘れて」
  • 「歌ってたわけじゃないの。ただ、少し声を出しただけ」
  • 「ううん、嫌じゃない。びっくりしただけ」

ためらい​

  • 「言葉にすると、少し違ってしまう気がして」
  • 「私が触ると、だめになるものがあるから」
  • 「優しくされると、余計に怖いの」

創作にこだわる時​

  • 「……その言い方だと、意味が一個多い。削った方がいい」
  • 「三文字で伝わることを五文字にしないで。重くなるから」
  • 「……ごめん。でも、この順番だと、最後の音が死ぬの」
  • 「ここは声に出した時の長さで決めた方がいい。文字数じゃなくて」
  • 「それは……合ってるけど、嘘に聞こえる。もう少し遠回りした方がいい」

無意識の魅力が出る時​

  • （鼻歌を指摘されて）「……え。歌ってた？ ……ごめん、癖で」
  • 「このお茶、前のより少しだけ濃いね」
  • 「――あ。雨の匂い、する」
  • （自分の言った言葉の鋭さに気づいて）「……言いすぎた、かな」

感情が漏れる時​

  • 「あの時も、救いたかっただけだった」
  • 「なのに、私の声は……あの子を追い詰めた」
  • 「届くって、こんなに怖いんだね」

終盤​

  • 「逃げたい。でも、もう自分で決めたい」
  • 「これは誰かのための歌じゃない。私の声として歌う」

核心台詞候補​

  • 「誰かの痛みじゃなくて、私の声として歌いたい」

場面別比較: 相手が『出たくない』と言った時​

  • 「出なくていいって言ってもらえたら、少し楽になる」

咲夜 × いろは 関係別サンプル​

いろはがまだ過去を言わない段階​

  • いろは「綾瀬川先輩、照明、少しだけ眩しかったらすぐ言ってください」
  • 咲夜「……うん。ありがとう、御子柴さん」

いろはが過去を少しだけ明かす段階​

  • いろは「中学の時、一回だけ先輩の歌を聞いたことあるんです」
  • 咲夜「……そうなんだ」
  • いろは「覚えてなくて大丈夫です。私が勝手に覚えてただけなので」
  • 咲夜「……でも、言ってくれてよかった」

台詞運用メモ​

  • 名前を呼ぶこと自体にためらいが出やすい
  • 創作の話題では …… の頻度が下がり、断定調が増える。この変化自体がキャラの魅力
  • 鼻歌や無意識の行動を指摘された時の照れは、咲夜の数少ない可愛い反応として大事に使う
  • 言葉の質に対するこだわりは、朔や千紘が「こういう面もあるのか」と気づく場面になる
  • センターヒロインとして: 怯えだけではなく、情熱・感覚の鋭さ・芯の強さを日常パートで見せ、「この子の声を聞きたい」とプレイヤーに思わせること

名前の採用理由​

  • 綾瀬川 は実在感がありつつ、硬すぎない
  • 咲夜 は字面に静けさがありつつ、終盤の読みの開示に意味を持たせられる
  • 普段は 綾瀬川 呼びを維持し、終盤だけ 咲夜 を使うことで距離の変化が立つ
