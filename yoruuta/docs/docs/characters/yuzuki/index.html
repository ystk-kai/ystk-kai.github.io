穂坂 結月（ほさか ゆづき）

基本情報​

  • 名前: 穂坂 結月（ほさか ゆづき）
  • 学年: 高校2年生
  • 立場: 別クラスの同級生、図書委員補助。湊とは中学の同級生

物語上の役割​

  • 一番静かな形で危うさを見せる
  • 企画の意味を問い直す

感情の核​

  • 諦め
  • 喪失感
  • 未来への期待の薄さ

家庭環境​

穂坂家​

  • 父: 穂坂 孝之: 結月が中学1年の時に離婚。現在は隣県に住んでいる。養育費は最初の1年は振り込まれていたが、次第に遅れがちになり、今はほぼ途絶えている。悪人ではないが、距離が物理的にも心理的にも開いた。結月は父を恨んでいるわけではなく、「いない人」として静かに処理している
  • 母: 穂坂 美弥子: 結月が中学2年の終わりに体調を崩し入院。退院後はスーパーのパートで生計を立てている。完治ではなく、疲れやすい体質が残っている。娘に負担をかけまいとするが、その気遣いが逆に結月を「心配させてはいけない」方向へ押している

家庭の経済状況​

  • 母のパート収入が主な収入源。養育費はあてにできない
  • 生活は回るが、余裕はない。結月が自分の欲しいものを言わなくなったのはこの頃から
  • 校外プログラムの参加費は奨学金で賄えたが、母の入院と重なり、家を空けること自体ができなかった
  • 結月は「お金がなかったから」とは言わない。「タイミングが合わなかった」と言う

家庭が結月に与えた影響​

  • 「欲しい」と言えない構造: 母が無理をして笑っているのを見てきた結月は、自分の望みを口にすること＝母に負担を増やすことだと学んだ。進路希望調査の白紙は怠惰ではなく、「書いたら誰かが困る」という判断
  • 優しさの檻: 美弥子は結月を責めない。結月も美弥子を責めない。互いに気を遣い合うことで、本音を言うタイミングが永遠に来ない。この穏やかさが二人を縛っている
  • 父の不在の影響: 恨みではなく「期待しても意味がない」の原型。父が去ったことで、結月は人が留まるという前提を持てなくなった。だから未来を信じることが構造的に難しい
  • 生活力の高さの理由: 母の入院中に覚えた家事・買い物・金銭管理が結月の「意外な知識」の正体。読書量は図書委員としてだけでなく、家で一人の時間が長かったから

咲夜・千紘との対比​

  • 咲夜: 経済的に困らない。だが声を出せない（内的な檻）
  • 千紘: 休めない。家が仕事場（構造的な檻）
  • 結月: 困っているが「困っている」と言えない（優しさの檻）
  • 三人とも自分で檻を作っている。だが檻の材質が全く違う

物語上の機能​

  • 「どっちでもいい」の根拠: 無気力ではない。どちらかを選ぶと、選ばなかった方で母が困るかもしれないから選べない
  • 朔への態度の背景: 結月が朔の善意を受け取れないのは、善意を受け取った後に失うことを知っているから。父がそうだった
  • Cルートの核心: 結月が「明日のリハーサルも来る」と自分から言えるようになることの重さ。それは「期待しても失う」の構造を、自分から壊しに行くこと
  • 些細なこだわりとの接続: 進路は決められない（母に影響する）が、ポン酢かケチャップかは決められる（誰も困らない）。この境界線が結月の現在地を正確に示す
  • 明示しすぎない: 母の体調は結月の生活描写（買い物の量、帰る時間、疲れた顔への反応）から推測させる。美弥子の直接登場は結月ルート内に限定

抱えている夜​

昨年、進学や受験につながる校外プログラムへの参加を、母親の入院と家計悪化のため断念した。 奨学金で費用は賄えたが、母の入院と重なり、家を空けること自体ができなかった。 誰も責めなかったが、それ以降結月は、

  • 期待してもどうせ失う
  • 自分の未来を考えること自体が贅沢だ
  • 欲しいと口にしたら、誰かが困る

と考えるようになった。

これにより、結月は強い絶望ではなく、静かな停止状態に入っている。

現在の状態​

  • 穏やかで協調的
  • だが進路にも企画にも、本気で手を伸ばしていない
  • 「どちらでもいい」と答える場面が増えている
  • 中学時代の友人とは今も話せるが、自分の将来に関わる話題だけは自然にかわす

性格​

  • 物腰が柔らかい
  • 感情の起伏を表に出しにくい
  • 気遣いはあるが、自分の意志を後回しにする
  • 否定より先に引いてしまう

止まる前の蓄積​

結月は「止まっている」が、止まる前の時間に蓄えたものは消えていない。 図書委員補助としての読書量、中学時代の活発だった記憶、母の入院以来身につけた生活の知恵。これらが不意に、本人も驚くくらい具体的な形で出る。

  • 結月本人はこれを「別に何でもない」として流す
  • 周囲が「え、なんでそれ知ってるの」と反応する
  • 止まっている ≠ 空っぽということを読者に刷り込む → 失われたらもったいないと感じさせる装置
  • 未来に関する知識・計画性は出さない。過去〜現在の蓄積だけ

些細なことへのこだわり​

進路や将来については「どっちでもいい」と言う結月が、些細なことに対しては意外なほど具体的な意見を持つ。

  • 食べ物の好み、本棚の並べ方、座る位置
  • これは結月が初めて能動的に「私はこうしたい」と言う場面になる
  • 進路は決められないが、ポン酢かケチャップかは決められる。このスケール感の落差が結月の現在地を示す
  • 小さなこだわりの積み重ねが、終盤の「大きな決断」への助走になる

見た目​

  • 身長160cm前後
  • 灰みのある栗色のゆるいロング、肩甲骨下まで
  • 普段は低めのひとつ結びか、ほどけかけた片寄せでまとめることが多い
  • 瞳はオリーブ寄りの淡い茶
  • 制服の上に淡いグレージュのカーディガン
  • 立ち姿は柔らかいが、重心が少し後ろに引いている
  • 配色はセージグリーン、グレージュ、薄琥珀

デザイン意図​

  • Aの直線、Bの鋭角に対して、Cは曖昧な曲線で構成する
  • 色味は一番淡くし、存在感が薄く見える危うさを演出する
  • 守ってあげたい印象より、「気づくと遠くにいる」印象を優先する
  • ロングヘア化してもAと被らないよう、髪の軽さと結び方の曖昧さで差を出す

一人称​

私（控えめで角のない印象を維持する）

口調​

  • 柔らかい
  • 否定をぼかす
  • 先に相手を気遣う
  • 追い詰められるほど、静かになる

呼称変化のイベント設計（結月ルート）​

  • 朔は序盤から 穂坂 と呼ぶ
  • 終盤で自然に 結月 へ移行する
  • 結月側は 久瀬くん を長く維持し、終盤で小さく 朔くん へ変えると効く

初登場​

  • 昼休み、湊の中学の同級生として朔の教室に来訪する
  • 感じの良い同級生として印象づく
  • 普通に応じるが、未来の話だけは自然に避ける

主人公との関係進展​

  • 感じの良い同級生
  • 何も期待しないことで傷を避けている相手
  • 未来を押しつけずに隣にいられる相手

台詞サンプル​

日常​

  • 「大丈夫。急がなくても、たぶん間に合うから」
  • 「私は、どっちでもいいよ。みんなが決めた方で」
  • 「楽しそうだね。……うん、本当に」
  • 「同じなら安心できることも、あるから」

意外な知識が出る時​

  • 「……それ、裏返して干した方がいいよ。色が抜けるから」
  • 「その話、前に読んだことある。結末は言わないけど、途中で一回怒ると思う」
  • 「……あ。このお菓子、前に湊が買ってたやつと同じ工場だと思う。裏のロット番号が近い」
  • 「ここ折ると綺麗にちぎれるよ。……母さんに教わった」

小さなこだわりが出る時​

  • 「……ごめん、これだけは。ハンバーグにはケチャップじゃなくてポン酢なの」
  • 「本棚は著者の五十音順じゃなくて、読んだ順に並べたい派」
  • 「この席がいい。……窓側じゃないと、少し落ち着かないから」

距離を取る時​

  • 「期待すると、あとで少し疲れるから」
  • 「別に諦めてるわけじゃないよ。ただ、考えないだけ」
  • 「頑張るのが嫌なんじゃないの。なくなった時がしんどいだけ」

本音が漏れる時​

  • 「私は何も失ってないって言われたら、それまでなんだけど」
  • 「でも、自分の番が来るって考えなくなったら、すごく楽だった」
  • 「楽っていうより……止まってるだけかもしれないけど」

終盤​

  • 「大きいことは言えない。でも、今日だけは逃げたくない」
  • 「明日を好きにはなれなくても、閉じないくらいならできるかも」

核心台詞候補​

  • 「明日を信じられなくても、閉じないことは選べる」

場面別比較: 相手が『出たくない』と言った時​

  • 「出なくても責めないよ。……私は、たぶん責められたくないだけだけど」

台詞運用メモ​

  • 会話の冒頭で相手の名前を置くと柔らかさが出る
  • 知識フラッシュは章あたり1回程度に絞る。頻繁だと「物知りキャラ」になり停滞感が消える
  • 周囲の「何で知ってるの」への結月の反応は「え、普通じゃない？」の温度で
  • 小さなこだわりは結月の数少ない能動的発言。日常パートで「この子は本当は豊かな人なのに」と思わせる装置
  • 湊の あかちゃん は雑な近さを示す常用呼称として使える
  • 結月の あかちゃん は昔の癖が漏れた時だけに限定すると効く
  • 結月は普段 あかりちゃん を選ぶことで、今の距離感と気遣いを見せる

名前の採用理由​

  • 穂坂 は静けさと距離感があり、結月の立ち位置に合う
  • 結月 は落ち着いた響きで、柔らかさを保ちつつ少し古風な品も出せる
  • ふわふわしすぎず、文学寄りに寄りすぎないバランスがある
